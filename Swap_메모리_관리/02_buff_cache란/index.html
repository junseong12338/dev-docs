02. buff/cacheлһҖВ¶

лӮңмқҙлҸ„: Alpha

01мһҘм—җм„ң "buff/cacheлҠ” л””мҠӨнҒ¬ мәҗмӢң"лқјкі л§Ң н–Ҳм–ҙ. мқҙлІҲм—” мҷң OSк°Җ к·ёкұё н•ҳлҠ”м§Җ, м–ём ң нҡҢмҲҳлҗҳлҠ”м§Җк№Ңм§Җ нҢҢкі л“ лӢӨ.

л№„мң : м№ҙнҺҳ л©”лүҙ мҷёмҡ°кё°В¶

мһҗмЈј к°ҖлҠ” м№ҙнҺҳк°Җ мһҲм–ҙ. л§ӨлІҲ л©”лүҙнҢҗ л“Өм—¬лӢӨліҙлҠ” кұҙ к·Җм°®мһ–м•„. к·ёлһҳм„ң мһҗмЈј мӢңнӮӨлҠ” кұё мҷёмӣҢлІ„л Ө. лӢӨмқҢл¶Җн„°лҠ” л©”лүҙнҢҗ м•Ҳ лҙҗлҸ„ л°”лЎң мЈјл¬ён•ҙ.

OSлҸ„ лҳ‘к°ҷм•„.

л№„мң  мӢӨм ң
л©”лүҙнҢҗ ліҙкё° л””мҠӨнҒ¬м—җм„ң мқҪкё° (лҠҗлҰј)
л©”лүҙ мҷёмҡ°кё° RAMм—җ мәҗмӢұ (buff/cache)
мҷёмҡҙ л©”лүҙлЎң мЈјл¬ё RAMм—җм„ң мқҪкё° (л№ лҰ„)
лЁёлҰ¬ мҡ©лҹү н•ңкі„ RAM мҡ©лҹү н•ңкі„

н•ң мӨ„ м •лҰ¬

buff/cacheлҠ” OSк°Җ л””мҠӨнҒ¬ лҚ°мқҙн„°лҘј RAMм—җ лҜёлҰ¬ мҳ¬л Өл‘” кІғмқҙм•ј. лӢӨмқҢм—җ к°ҷмқҖ лҚ°мқҙн„° н•„мҡ”н•ҳл©ҙ л””мҠӨнҒ¬ м•Ҳ к°Җкі  RAMм—җм„ң л°”лЎң к°Җм ёмҷҖ.

мҷң мәҗмӢңлҘј н•ҳлҠ”к°Җ: мҶҚлҸ„ м°ЁмқҙВ¶

мқҙкІҢ м ңмқј мӨ‘мҡ”н•ҙ. л””мҠӨнҒ¬лһ‘ RAMмқҳ мҶҚлҸ„ м°Ёмқҙк°Җ м–јл§ҲлӮҳ лӮҳлҠ”м§Җ лҙҗ.

м ҖмһҘмһҘм№ҳлі„ мҶҚлҸ„ л№„көҗВ¶

м ҖмһҘмһҘм№ҳ мқҪкё° мҶҚлҸ„ м ‘к·ј мӢңк°„ л№„мң 
HDD ~200 MB/s ~10ms лҸ„м„ңкҙҖ к°Җм„ң мұ… м°ҫкё°
SSD (SATA) ~550 MB/s ~0.1ms мұ…мғҒ м„ңлһҚм—җм„ң кәјлӮҙкё°
SSD (NVMe) ~3,500 MB/s ~0.02ms мҶҗм—җ л“Өкі  мһҲлҠ” л…ёнҠё
RAM ~50,000 MB/s ~100ns мқҙлҜё лЁёлҰҝмҶҚм—җ мһҲлҠ” кІғ

н•өмӢ¬ мҲ«мһҗ

RAMмқҖ HDDліҙлӢӨ м•Ҫ 10л§Ң л°° л№Ёлқј. NVMe SSDлһ‘ л№„көҗн•ҙлҸ„ м•Ҫ 15л°° л№Ёлқј.

  • кІ°лЎ : мһҗмЈј мқҪлҠ” нҢҢмқјмқјмҲҳлЎқ м„ұлҠҘ мқҙл“қмқҙ м»Ө
