мқҙкұ° мӢӨм ңлЎң л°ңмғқн•ҳлҠ” л¬ём ңлӢӨ

Nginxк°Җ мҡ”мІӯмқ„ м–ҙл–Ө WASлЎң ліҙлӮјм§ҖлҠ” к·ёл•Ңк·ёл•Ң лӢӨлҘҙлӢӨ (лқјмҡҙл“ңлЎңл№Ҳ, мөңмҶҢм—°кІ° л“ұ). н•ң лІҲмқҖ WAS01, лӢӨмқҢм—җлҠ” WAS02лЎң к°Ҳ мҲҳ мһҲлӢӨ. л§ӨлІҲ к°ҷмқҖ м„ңлІ„лЎң к°ҖлҠ” ліҙмһҘмқҙ м—ҶлӢӨ.

н•ҙкІ° л°©лІ•л“ӨВ¶

л°©лІ• м„ӨлӘ… лӢЁм җ
Sticky Session к°ҷмқҖ мӮ¬мҡ©мһҗлҠ” н•ӯмғҒ к°ҷмқҖ м„ңлІ„лЎң ліҙлғ„ м„ңлІ„ н•ҳлӮҳ мЈҪмңјл©ҙ м„ём…ҳ лӮ м•„к°җ
Session Replication м„ңлІ„ к°„ м„ём…ҳ ліөм ң м„ңлІ„ лҠҳмҲҳлЎқ ліөм ң л¶Җн•ҳ мҰқк°Җ
мҷёл¶Җ м„ём…ҳ м ҖмһҘмҶҢ (Redis) м„ём…ҳмқ„ лі„лҸ„ м„ңлІ„м—җ м ҖмһҘ Redis м„ңлІ„ кҙҖлҰ¬ н•„мҡ”

мҡ°лҰ¬ н”„лЎңм қнҠёмқҳ м„ нғқ: Redis

RedisлҘј мҷёл¶Җ м„ём…ҳ м ҖмһҘмҶҢлЎң мӮ¬мҡ©н•ңлӢӨ. лӘЁл“  WASк°Җ к°ҷмқҖ RedisлҘј л°”лқјліҙлӢҲк№Ң, м–ҙл–Ө WASлЎң мҡ”мІӯмқҙ к°Җл“  к°ҷмқҖ м„ём…ҳмқ„ м“ё мҲҳ мһҲлӢӨ.

RedisлһҖ?В¶

Redis (Remote Dictionary Server)

  • мқёл©”лӘЁлҰ¬ Key-Value м ҖмһҘмҶҢ: лҚ°мқҙн„°лҘј л©”лӘЁлҰ¬м—җ м ҖмһҘн•ҙм„ң м—„мІӯ л№ лҘҙлӢӨ
  • лі„лҸ„мқҳ м„ңлІ„: Tomcat м•Ҳм—җ мһҲлҠ” кІҢ м•„лӢҲлӢӨ. мҷ„м „нһҲ лҸ…лҰҪлҗң м„ңлІ„ н”„лЎңм„ёмҠӨ
  • Key-Value кө¬мЎ°: нӮӨлЎң к°’мқ„ м°ҫлҠ” л”•м…”л„ҲлҰ¬(Map) кө¬мЎ°

RedisлҠ” лҚ°мқҙн„°лІ мқҙмҠӨ(MySQL л“ұ)мҷҖ лӢӨлҘҙлӢӨ:

кө¬л¶„ MySQL Redis
м ҖмһҘ мң„м№ҳ л””мҠӨнҒ¬ (HDD/SSD) л©”лӘЁлҰ¬ (RAM)
мҶҚлҸ„ мғҒлҢҖм ҒмңјлЎң лҠҗлҰј л§Өмҡ° л№ лҰ„ (ms мқҙн•ҳ)
лҚ°мқҙн„° кө¬мЎ° н…Ңмқҙлё”, н–ү, м—ҙ Key-Value
мҡ©лҸ„ мҳҒкө¬ лҚ°мқҙн„° м ҖмһҘ мәҗмӢң, м„ём…ҳ, мһ„мӢң лҚ°мқҙн„°
лҚ°мқҙн„° мҶҢл©ё м§Ғм ‘ мӮӯм ңн•ҳкё° м „к№Ңм§Җ мң м§Җ expire мӢңк°„ нӣ„ мһҗлҸҷ мӮӯм ң

Redisм—җ м„ём…ҳмқҙ мқҙл ҮкІҢ м ҖмһҘлҗңлӢӨ:

TTL (Time To Live)

Redisмқҳ expire м„Өм •. мқҙ мӢңк°„мқҙ м§ҖлӮҳл©ҙ Redisк°Җ мһҗлҸҷмңјлЎң нӮӨлҘј мӮӯм ңн•ңлӢӨ. мҡ°лҰ¬ н”„лЎңм қнҠём—җм„ңлҠ” 120л¶„(7200мҙҲ)мңјлЎң м„Өм •лҗҳм–ҙ мһҲлӢӨ. web.xmlмқҳ session-timeoutкіј л§һм¶ҳ кұ°лӢӨ.

HUBмҷҖ LMSмқҳ м„ём…ҳ кіөмң В¶

м—¬кё°м„ң мӨ‘мҡ”н•ң кІҢ н•ҳлӮҳ лҚ” мһҲлӢӨ. мҡ°лҰ¬ мӢңмҠӨн…ңмқҖ HUB(KNU10WebService)мҷҖ LMS(LXP-KNU10)к°Җ лі„лҸ„ м„ңлІ„лӢӨ.

sequenceDiagram
    participant B as лёҢлқјмҡ°м Җ
    participant H as HUB м„ңлІ„

    participant L as LMS м„ңлІ„

    B->>H: 1. лЎңк·ёмқё мҡ”мІӯ
    H->>R: 2. м„ём…ҳ м ҖмһҘ (UserNo, UserType...)
    R->>H: OK

    B->>L: 4. LMS м ‘мҶҚ (Cookie: JSESSIONID=abc123)

    R->>L: 6. UserNo=20210001, UserType=STUDENT

  1. мӮ¬мҡ©мһҗк°Җ HUBм—җм„ң лЎңк·ёмқён•ңлӢӨ
  2. HUBк°Җ Redisм—җ м„ём…ҳ лҚ°мқҙн„°лҘј м ҖмһҘн•ңлӢӨ
  3. лёҢлқјмҡ°м Җк°Җ JSESSIONID мҝ нӮӨлҘј л°ӣлҠ”лӢӨ
  4. мӮ¬мҡ©мһҗк°Җ LMSм—җ м ‘мҶҚн•ңлӢӨ (к°ҷмқҖ мҝ нӮӨлҘј ліҙлғ„)
  5. LMSк°Җ Redisм—җм„ң м„ём…ҳмқ„ мЎ°нҡҢн•ңлӢӨ
  6. Redisк°Җ HUBм—җм„ң м ҖмһҘн•ң м„ём…ҳ лҚ°мқҙн„°лҘј лҸҢл ӨмӨҖлӢӨ
  7. LMSлҸ„ мқҙ мӮ¬мҡ©мһҗлҘј лЎңк·ёмқёлҗң мӮ¬мҡ©мһҗлЎң мқёмӢқн•ңлӢӨ

Redisк°Җ м—Ҷмңјл©ҙ?

HUBмҷҖ LMSлҠ” м„ңлЎң лӢӨлҘё м„ңлІ„лӢӨ. HUB л©”лӘЁлҰ¬мқҳ м„ём…ҳмқ„ LMSк°Җ ліј мҲҳ м—ҶлӢӨ. HUBм—җм„ң лЎңк·ёмқён–ҲлҠ”лҚ° LMSм—җм„ң "лЎңк·ёмқён•ҳм„ёмҡ”"к°Җ лң¬лӢӨ. Redisк°Җ л‘җ мӢңмҠӨн…ңмқ„ м—°кІ°н•ҳлҠ” лӢӨлҰ¬ м—ӯн• мқ„ н•ңлӢӨ.

мҡ°лҰ¬ н”„лЎңм қнҠё м„Өм •В¶

# Redis м„Өм •
redis.host={REDIS_SERVER_IP}
redis.port=6379
redis.expire=120
м„Өм • к°’ мқҳлҜё
redis.host {REDIS_SERVER_IP} Redis м„ңлІ„ IP
redis.port 6379 Redis кё°ліё нҸ¬нҠё
redis.expire 120 м„ём…ҳ л§ҢлЈҢ мӢңк°„ (л¶„)

RedisлҠ” лі„лҸ„ м„ңлІ„

{REDIS_SERVER_IP}лҠ” WASмҷҖ лӢӨлҘё м„ңлІ„лӢӨ. RedisлҠ” Tomcat м•Ҳм—җ лӮҙмһҘлҗң кІҢ м•„лӢҲлқј, лі„лҸ„лЎң м„Өм№ҳлҗң лҸ…лҰҪ н”„лЎңм„ёмҠӨлӢӨ. WASк°Җ л„ӨнҠёмӣҢнҒ¬лҘј нҶөн•ҙ Redisм—җ м ‘мҶҚн•ңлӢӨ.

м„ём…ҳ л§ҢлЈҢ мӢң мқјм–ҙлӮҳлҠ” мқјВ¶

sequenceDiagram

    participant W as WAS (Tomcat)
    participant R as Redis

    Note over B,R: 120л¶„ лҸҷм•Ҳ мҡ”мІӯ м—ҶмқҢ
    R->>R: TTL л§ҢлЈҢ вҶ’ м„ём…ҳ лҚ°мқҙн„° мӮӯм ң

    B->>W: нҺҳмқҙм§Җ мҡ”мІӯ (Cookie: JSESSIONID=abc123)
    W->>R: abc123 м„ём…ҳ мЎ°нҡҢ
    R->>W: null (м—ҶмқҢ)

  2. мӮ¬мҡ©мһҗк°Җ лӢӨмӢң м ‘мҶҚн•ҳл©ҙ WASк°Җ Redisм—җм„ң м„ём…ҳмқ„ м°ҫмңјл Ө н•ңлӢӨ
  3. Redisм—җ м—ҶлӢӨ вҶ’ UserBroker.getUserNo()к°Җ л№Ҳ л¬ёмһҗм—ҙ л°ҳнҷҳ
  4. SecurityUtil.authorizationCheck()м—җм„ң л№Ҳ л¬ёмһҗм—ҙ к°җм§Җ вҶ’ SessionBrokenException throw
  5. лЎңк·ёмқё нҺҳмқҙм§ҖлЎң лҰ¬лӢӨмқҙл үнҠё

мқҙкІҢ м •мғҒ нқҗлҰ„мқҙлӢӨ

м„ём…ҳ л§ҢлЈҢ вҶ’ лЎңк·ёмқё лҰ¬лӢӨмқҙл үнҠёлҠ” м •мғҒм Ғмқё лҸҷмһ‘мқҙлӢӨ. м—җлҹ¬к°Җ м•„лӢҲлӢӨ. к·јлҚ° мҡ°лҰ¬ мҪ”л“ңлҠ” мқҙкұё e.printStackTrace()лЎң м°Қкі  мһҲлӢӨ. мқҙкІҢ мҷң л¬ём ңмқём§ҖлҠ” 08мһҘм—җм„ң.

н•өмӢ¬ м •лҰ¬В¶

  1. м„ңлІ„к°Җ м—¬лҹ¬ лҢҖл©ҙ л©”лӘЁлҰ¬ м„ём…ҳмқҙ кіөмң  м•Ҳ лҗЁ вҶ’ RedisлЎң н•ҙкІ°
  2. Redis = лі„лҸ„ м„ңлІ„м—җм„ң лҸҢм•„к°ҖлҠ” мқёл©”лӘЁлҰ¬ Key-Value м ҖмһҘмҶҢ
  3. HUBм—җм„ң лЎңк·ёмқё вҶ’ Redisм—җ м„ём…ҳ м ҖмһҘ вҶ’ LMSм—җм„ң Redis мЎ°нҡҢ вҶ’ лЎңк·ёмқё мң м§Җ
  4. expire=120л¶„, TTL л§ҢлЈҢ мӢң Redisк°Җ мһҗлҸҷ мӮӯм ң
  5. м„ём…ҳ л§ҢлЈҢ нӣ„ м ‘мҶҚ вҶ’ UserBroker л№Ҳ к°’ вҶ’ SessionBrokenException вҶ’ лЎңк·ёмқё лҰ¬лӢӨмқҙл үнҠё

нҷ•мқёл¬ём ңВ¶

Q1. RedisлҘј мӮ¬мҡ©н•ҳлҠ” мқҙмң В¶

л¬ём ң

м„ём…ҳ кіөмң  л¬ём ңлҘј н•ҙкІ°н•ҳлҠ” л°©лІ•мңјлЎң Sticky SessionлҸ„ мһҲлҠ”лҚ°, мҷң RedisлҘј м„ нғқн–Ҳмқ„к№Ң? Sticky Sessionмқҳ лӢЁм җмқ„ 2к°Җм§Җ л§җн•ҙлҙҗ.

м •лӢө ліҙкё°
  1. м„ңлІ„ мһҘм•  мӢң м„ём…ҳ мң мӢӨ: WAS01м—җ л¶ҷм–ҙмһҲлҚҳ мӮ¬мҡ©мһҗмқҳ м„ём…ҳмқҙ WAS01 л©”лӘЁлҰ¬м—җл§Ң мһҲмңјлҜҖлЎң, WAS01мқҙ мЈҪмңјл©ҙ м„ём…ҳмқҙ м „л¶Җ лӮ м•„к°„лӢӨ. лӢӨмӢң лЎңк·ёмқён•ҙм•ј н•ңлӢӨ.
  2. л¶Җн•ҳ л¶„мӮ° л¶Ҳк· нҳ•: нҠ№м • м„ңлІ„м—җ м„ём…ҳмқҙ л§Һмқҙ лӘ°лҰ¬л©ҙ к·ё м„ңлІ„л§Ң кіјл¶Җн•ҳк°Җ кұёлҰ°лӢӨ. лӢӨлҘё м„ңлІ„к°Җ м—¬мң  мһҲм–ҙлҸ„ мқҙлҸҷмӢңнӮ¬ мҲҳ м—ҶлӢӨ.

RedisлҠ” лі„лҸ„ м„ңлІ„мқҙлҜҖлЎң WASк°Җ мЈҪм–ҙлҸ„ м„ём…ҳмқҖ мӮҙм•„мһҲкі , м–ҙл–Ө WASл“  Redisм—җм„ң к°ҷмқҖ м„ём…ҳмқ„ к°Җм ёмҳ¬ мҲҳ мһҲлӢӨ.

Q2. Redis лҚ°мқҙн„° мҶҢл©ёВ¶

л¬ём ң

Redisм—җ м ҖмһҘлҗң м„ём…ҳмқҙ мӮӯм ңлҗҳлҠ” кІҪмҡ° 2к°Җм§ҖлҘј л§җн•ҙлҙҗ.

м •лӢө ліҙкё°
  1. TTL(expire) л§ҢлЈҢ: м„Өм •лҗң мӢңк°„(120л¶„) лҸҷм•Ҳ к°ұмӢ мқҙ м—Ҷмңјл©ҙ Redisк°Җ мһҗлҸҷмңјлЎң мӮӯм ңн•ңлӢӨ.
  2. лЎңк·ём•„мӣғ: session.invalidate()к°Җ нҳём¶ңлҗҳл©ҙ Redisм—җм„ңлҸ„ н•ҙлӢ№ м„ём…ҳ нӮӨк°Җ мӮӯм ңлҗңлӢӨ.

м¶”к°ҖлЎң Redis м„ңлІ„ мһҗмІҙк°Җ мһ¬мӢңмһ‘лҗҳл©ҙ л©”лӘЁлҰ¬мқҳ лҚ°мқҙн„°к°Җ лӮ м•„к°Ҳ мҲҳ мһҲлӢӨ (persistence м„Өм •м—җ л”°лқј лӢӨлҰ„).

Q3. HUBмҷҖ LMS м„ём…ҳ кіөмң В¶

л¬ём ң

мӮ¬мҡ©мһҗк°Җ HUBм—җм„ң лЎңк·ёмқён•ң нӣ„ LMSм—җ м ‘мҶҚн–ҲлҠ”лҚ° "лЎңк·ёмқён•ҳм„ёмҡ”"к°Җ л–ҙлӢӨ. мӣҗмқёмңјлЎң к°ҖлҠҘн•ң кІғ 3к°Җм§ҖлҘј м¶”лЎ н•ҙлҙҗ.

м •лӢө ліҙкё°
  1. Redis м„ңлІ„ мһҘм• : Redisк°Җ лӢӨмҡҙлҗҳл©ҙ LMSк°Җ м„ём…ҳмқ„ мЎ°нҡҢн•  мҲҳ м—ҶлӢӨ.
  2. м„ём…ҳ л§ҢлЈҢ: HUB лЎңк·ёмқё нӣ„ 120л¶„мқҙ м§ҖлӮҳм„ң Redisм—җм„ң м„ём…ҳмқҙ мӮӯм ңлҗҗлӢӨ.
  3. мҝ нӮӨ л¬ём ң: HUBмҷҖ LMSмқҳ лҸ„л©”мқёмқҙ лӢ¬лқјм„ң лёҢлқјмҡ°м Җк°Җ мҝ нӮӨлҘј м•Ҳ ліҙлғҲлӢӨ. мҝ нӮӨмқҳ domain м„Өм •мқҙ л‘җ м„ңлІ„лҘј нҸ¬н•Ён•ҳм§Җ м•Ҡмңјл©ҙ мқҙлҹ° мқјмқҙ мғқкёҙлӢӨ.

Q4. Redis vs лҚ°мқҙн„°лІ мқҙмҠӨВ¶

л¬ём ң

м„ём…ҳмқ„ Redisк°Җ м•„лӢҲлқј MySQL(лҚ°мқҙн„°лІ мқҙмҠӨ)м—җ м ҖмһҘн•ҳл©ҙ м•Ҳ лҗҳлӮҳ? лҗҳкёҙ лҗңлӢӨл©ҙ, RedisлҘј м“°лҠ” мқҙмң лҠ”?

м •лӢө ліҙкё°

MySQLм—җ м ҖмһҘн•ҙлҸ„ лҸҷмһ‘мқҖ н•ңлӢӨ. н•ҳм§Җл§Ң RedisлҘј м“°лҠ” мқҙмң лҠ” мҶҚлҸ„ л•Ңл¬ёмқҙлӢӨ.

м„ём…ҳ мЎ°нҡҢлҠ” л§Ө мҡ”мІӯл§ҲлӢӨ л°ңмғқн•ңлӢӨ. мқён„°м…үн„°к°Җ л§ӨлІҲ м„ём…ҳмқ„ нҷ•мқён•ҳлӢҲк№Ң. MySQLмқҖ л””мҠӨнҒ¬ I/Oк°Җ н•„мҡ”н•ҙм„ң 1-10ms м •лҸ„ кұёлҰ¬м§Җл§Ң, RedisлҠ” л©”лӘЁлҰ¬м—җм„ң л°”лЎң мқҪмңјлҜҖлЎң 0.1ms мқҙн•ҳлӢӨ.

мҡ”мІӯмқҙ мҙҲлӢ№ 1000к°ңл©ҙ: MySQL = мҙҲлӢ№ 1000лІҲ л””мҠӨнҒ¬ м ‘к·ј vs Redis = мҙҲлӢ№ 1000лІҲ л©”лӘЁлҰ¬ м ‘к·ј. м°Ёмқҙк°Җ м—„мІӯлӮҳлӢӨ.

Q5. expire м„Өм •В¶

л¬ём ң

redis.expire=120кіј web.xmlмқҳ session-timeout=120мқҙ к°ҷмқҖ к°’(120л¶„)мқё мқҙмң к°Җ лӯҗм•ј? лӢӨлҘё к°’мқҙл©ҙ м–ҙл–»кІҢ лҗҳлӮҳ?

м •лӢө ліҙкё°

л‘җ к°’мқҙ к°ҷм•„м•ј м„ём…ҳ л§ҢлЈҢ лҸҷмһ‘мқҙ мқјкҙҖм Ғмқҙкё° л•Ңл¬ёмқҙлӢӨ.

л§Ңм•Ҫ Redis expire=60л¶„, web.xml timeout=120л¶„мқҙл©ҙ: - 60л¶„ нӣ„ Redisм—җм„ң м„ём…ҳ мӮӯм ңлҗЁ - TomcatмқҖ м•„м§Ғ 120л¶„ м•Ҳ м§ҖлӮ¬лӢӨкі  мғқк°Ғн•Ё - кІ°кіј: Tomcatмқҳ м„ём…ҳмқҖ мӮҙм•„мһҲлҠ”лҚ° Redis лҚ°мқҙн„°лҠ” м—ҶмқҢ вҶ’ нҳјлһҖ

л°ҳлҢҖлЎң Redis expire=120л¶„, web.xml timeout=60л¶„мқҙл©ҙ: - 60л¶„ нӣ„ Tomcat м„ём…ҳ л§ҢлЈҢ - Redisм—җлҠ” м•„м§Ғ лҚ°мқҙн„°к°Җ лӮЁм•„мһҲмқҢ вҶ’ л©”лӘЁлҰ¬ лӮӯл№„

л‘җ к°’мқ„ лҸҷмқјн•ҳкІҢ л§һм¶°м•ј к№”лҒ”н•ҳлӢӨ.
