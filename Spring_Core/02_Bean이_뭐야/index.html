02. Beanмқҙ лӯҗм•ј¶

мқҙ мһҘмқҳ лӘ©н‘ң: "Beanмқҙ лӯҗм•ј?"м—җ м •нҷ•нһҲ лҢҖлӢөн•ҳкі , XMLм—җм„ң Bean л“ұлЎқн•ҳлҠ” кұё мқҪмқ„ мҲҳ мһҲлӢӨ

1. Beanмқҳ м •мқҳ¶

1.1 н•ң л¬ёмһҘ м •мқҳ¶

Bean = Spring м»Ён…Ңмқҙл„Ҳк°Җ мғқм„ұн•ҳкі  кҙҖлҰ¬н•ҳлҠ” к°қмІҙ

к·ёлғҘ "к°қмІҙ"м•ј. нҠ№лі„н•ң кІҢ м•„лӢҲлӢӨ.
лӢӨл§Ң "Springмқҙ кҙҖлҰ¬н•ҳлҠ”" к°қмІҙлқјлҠ” м җмқҙ лӢӨлҘҙлӢӨ.

1.2 мқјл°ҳ к°қмІҙ vs Bean¶

// мқјл°ҳ к°қмІҙ - лӮҙк°Җ м§Ғм ‘ л§Ңл“ лӢӨ
UserService service = new UserService();
// вҶ’ мқҙкұҙ Beanмқҙ м•„лӢҲлӢӨ
// вҶ’ лӮҙк°Җ л§Ңл“Өм—Ҳкі , лӮҙк°Җ кҙҖлҰ¬н•ңлӢӨ

// Bean - Springмқҙ л§Ңл“ лӢӨ
// XMLм—җ мқҙл ҮкІҢ м„ м–ён•ҳл©ҙ:
// <bean id="userService" class="com.example.UserService" />
// вҶ’ Springмқҙ UserService к°қмІҙлҘј л§Ңл“ лӢӨ
// вҶ’ Springмқҙ кҙҖлҰ¬н•ңлӢӨ
// вҶ’ мқҙкІҢ BeanмқҙлӢӨ

1.3 н•өмӢ¬ м°Ёмқҙ¶

мқјл°ҳ к°қмІҙ vs Bean

мқјл°ҳ к°қмІҙ:

  • лӮҙк°Җ newлЎң л§Ңл“ лӢӨ
  • лӮҙк°Җ мӮ¬мҡ©н•ңлӢӨ
  • GCк°Җ м•Ңм•„м„ң м •лҰ¬н•ңлӢӨ
  • лҲ„к°Җ мқҙ к°қмІҙлҘј м“°лҠ”м§Җ м•„л¬ҙлҸ„ лӘЁлҘёлӢӨ

Bean:

  • Springмқҙ л§Ңл“ лӢӨ
  • Springмқҙ мқҙлҰ„(id)мқ„ л¶ҷм—¬м„ң кҙҖлҰ¬н•ңлӢӨ
  • Springмқҙ мғқлӘ…мЈјкё°лҘј кҙҖлҰ¬н•ңлӢӨ (мғқм„ұвҶ’мҙҲкё°нҷ”вҶ’мӮ¬мҡ©вҶ’мҶҢл©ё)
  • лҲ„к°Җ мқҙ Beanмқ„ м“°лҠ”м§Җ Springмқҙ м•ҲлӢӨ
  • н•„мҡ”н•ң кіім—җ Springмқҙ л„Јм–ҙмӨҖлӢӨ (DI)

2. Bean л“ұлЎқ л°©лІ•¶

2.1 XMLлЎң л“ұлЎқ (мҡ°лҰ¬ н”„лЎңм қнҠё л°©мӢқ)¶

<!-- к°ҖмһҘ кё°ліём Ғмқё Bean л“ұлЎқ -->
<bean id="userService" class="com.example.service.UserService" />

н•ҙм„қ:
  <bean          вҶ’ "Beanмқ„ л“ұлЎқн•ҳкІ лӢӨ"
  id="userService"   вҶ’ "мқҙлҰ„мқҖ userServiceлЎң"
  class="com.example.service.UserService"  вҶ’ "мқҙ нҒҙлһҳмҠӨмқҳ к°қмІҙлҘј"
  />             вҶ’ "лҒқ"

Springмқҙ н•ҳлҠ” мқј:
  UserService userService = new UserService();
  м»Ён…Ңмқҙл„Ҳм—җ "userService"лқјлҠ” мқҙлҰ„мңјлЎң м ҖмһҘ

2.2 idлһҖ¶

id = Beanмқҳ кі мң  мқҙлҰ„
  вҶ’ м»Ён…Ңмқҙл„Ҳм—җм„ң мқҙ Beanмқ„ м°ҫмқ„ л•Ң м“°лҠ” мӢқлі„мһҗ
  вҶ’ мӨ‘ліө л¶Ҳк°Җ

л№„мң :
  м•„нҢҢнҠё нҳёмҲҳмҷҖ к°ҷлӢӨ
  "302нҳё мЈјлҜј л¶Ҳлҹ¬мЈјм„ёмҡ”" = "userService Bean к°Җм ёмҷҖ"

мҳҲмӢң:
  <bean id="sqlSession" class="org.mybatis.spring.SqlSessionFactoryBean" />
  вҶ’ "sqlSession"мқҙлқјлҠ” мқҙлҰ„мңјлЎң SqlSessionFactoryBean к°қмІҙлҘј л“ұлЎқ

2.3 classлһҖ¶

class = м–ҙл–Ө нҒҙлһҳмҠӨмқҳ к°қмІҙлҘј л§Ңл“Ө кұҙм§Җ
  вҶ’ нҢЁнӮӨм§ҖлӘ… нҸ¬н•Ё м „мІҙ кІҪлЎң (FQCN: Fully Qualified Class Name)

мҳҲмӢң:
  class="org.mybatis.spring.SqlSessionFactoryBean"

  нҢЁнӮӨм§Җ: org.mybatis.spring
  нҒҙлһҳмҠӨ: SqlSessionFactoryBean

  SpringмқҖ мқҙкұё ліҙкі :
  вҶ’ мқҙ нҒҙлһҳмҠӨлҘј м°ҫм•„м„ң
  вҶ’ new SqlSessionFactoryBean() мқ„ мӢӨн–үн•ңлӢӨ

3. Beanм—җ к°’ л„Јкё° (property)¶

3.1 value - лӢЁмҲң к°’ л„Јкё°¶

<bean id="dataSource" class="org.apache.commons.dbcp2.BasicDataSource">
    <property name="url" value="jdbc:mariadb://localhost:3306/mydb" />
    <property name="username" value="root" />
    <property name="password" value="1234" />
</bean>

н•ҙм„қ:
  <property          вҶ’ "мқҙ Beanмқҳ н•„л“ңм—җ к°’мқ„ л„ЈкІ лӢӨ"
  name="url"         вҶ’ "urlмқҙлқјлҠ” н•„л“ңм—җ"
  value="jdbc:..."   вҶ’ "мқҙ л¬ёмһҗм—ҙ к°’мқ„"
  />

Springмқҙ н•ҳлҠ” мқј:
  BasicDataSource dataSource = new BasicDataSource();
  dataSource.setUrl("jdbc:mariadb://localhost:3306/mydb");
  dataSource.setUsername("root");
  dataSource.setPassword("1234");

н•өмӢ¬:
  name="url" вҶ’ setUrl() л©”м„ңл“ңлҘј нҳём¶ңн•ңлӢӨ
  name="username" вҶ’ setUsername() л©”м„ңл“ңлҘј нҳём¶ңн•ңлӢӨ
  вҶ’ propertyмқҳ nameмқҖ setter л©”м„ңл“ң мқҙлҰ„кіј л§Өн•‘лҗңлӢӨ

3.2 ref - лӢӨлҘё Bean м°ёмЎ°¶

<bean id="dataSource" class="org.apache.commons.dbcp2.BasicDataSource">
    <property name="url" value="jdbc:mariadb://..." />
</bean>

<bean id="sqlSession" class="org.mybatis.spring.SqlSessionFactoryBean">
    <property name="dataSource" ref="dataSource" />
</bean>

н•ҙм„қ:
  <property name="dataSource" ref="dataSource" />

  name="dataSource"  вҶ’ "мқҙ Beanмқҳ dataSource н•„л“ңм—җ"
  ref="dataSource"   вҶ’ "dataSourceлқјлҠ” мқҙлҰ„мқҳ лӢӨлҘё Beanмқ„ л„Јм–ҙлқј"

  value vs ref:
    value = л¬ёмһҗм—ҙ, мҲ«мһҗ л“ұ лӢЁмҲң к°’
    ref   = лӢӨлҘё Bean к°қмІҙлҘј м°ёмЎ°

Springмқҙ н•ҳлҠ” мқј:
  // лЁјм Җ dataSource Beanмқ„ л§Ңл“ лӢӨ
  BasicDataSource dataSource = new BasicDataSource();
  dataSource.setUrl("jdbc:mariadb://...");

  // к·ё лӢӨмқҢ sqlSession Beanмқ„ л§Ңл“Өкі , dataSourceлҘј л„Јм–ҙмӨҖлӢӨ
  SqlSessionFactoryBean sqlSession = new SqlSessionFactoryBean();
  sqlSession.setDataSource(dataSource);  // ref="dataSource" вҶ’ мқҙ мӨ„

мқҙкІҢ л°”лЎң DI(мқҳмЎҙм„ұ мЈјмһ…)лӢӨ.
sqlSessionмқҙ dataSourceлҘј "н•„мҡ”"лЎң н•ҳлҠ”лҚ°,
Springмқҙ "м•Ңм•„м„ң л„Јм–ҙмӨҖ" кІғ.

3.3 value vs ref м •лҰ¬¶

value vs ref

value: вҶ’ л¬ёмһҗм—ҙ, мҲ«мһҗ, boolean л“ұ лӢЁмҲң к°’ вҶ’ value="1000" вҶ’ value="jdbc:mariadb://..." вҶ’ value="true"

ref: вҶ’ лӢӨлҘё Bean к°қмІҙлҘј м°ёмЎ° вҶ’ ref="dataSource" вҶ’ dataSourceлқјлҠ” idмқҳ Beanмқ„ л„Јм–ҙлқј вҶ’ ref="sqlSession" вҶ’ sqlSessionмқҙлқјлҠ” idмқҳ Beanмқ„ л„Јм–ҙлқј

н—·к°ҲлҰ¬л©ҙ: "л„Јмқ„ кІҢ кёҖмһҗ/мҲ«мһҗм•ј?" вҶ’ value "л„Јмқ„ кІҢ лӢӨлҘё к°қмІҙм•ј?" вҶ’ ref

4. мҡ°лҰ¬ н”„лЎңм қнҠём—җм„ң Bean мқҪкё°¶

4.1 context-sqlMap.xml¶

<bean id="sqlSession" class="org.mybatis.spring.SqlSessionFactoryBean">
    <property name="dataSource" ref="dataSource" />
    <property name="configLocation"
              value="classpath:/egovframework/sqlmap/sql-mapper-config.xml" />
    <property name="mapperLocations"
              value="classpath*:/egovframework/sqlmap/${framework.database.db_type}/*/*.xml" />
</bean>

н•ң мӨ„м”© н•ҙм„қ:

1. <bean id="sqlSession" class="org.mybatis.spring.SqlSessionFactoryBean">
   вҶ’ "sqlSession"мқҙлқјлҠ” мқҙлҰ„мңјлЎң SqlSessionFactoryBean к°қмІҙлҘј л§Ңл“Өм–ҙлқј

2. <property name="dataSource" ref="dataSource" />
   вҶ’ dataSource н•„л“ңм—җ "dataSource"лқјлҠ” Beanмқ„ л„Јм–ҙлқј (ref = лӢӨлҘё Bean)

3. <property name="configLocation" value="classpath:/.../sql-mapper-config.xml" />
   вҶ’ configLocation н•„л“ңм—җ мқҙ кІҪлЎң л¬ёмһҗм—ҙмқ„ л„Јм–ҙлқј (value = л¬ёмһҗм—ҙ)

4. <property name="mapperLocations" value="classpath*:/.../maria/*/*.xml" />
   вҶ’ mapperLocations н•„л“ңм—җ мқҙ кІҪлЎң нҢЁн„ҙмқ„ л„Јм–ҙлқј (value = л¬ёмһҗм—ҙ)
   вҶ’ classpath*: вҶ’ лӘЁл“  classpathм—җм„ң м°ҫкІ лӢӨ
   вҶ’ */*.xml вҶ’ н•ҳмң„нҸҙлҚ”/м•„л¬ҙмқҙлҰ„.xml

Springмқҙ мӢӨм ңлЎң н•ҳлҠ” кІғ:
  SqlSessionFactoryBean sqlSession = new SqlSessionFactoryBean();
  sqlSession.setDataSource(dataSource);                    // ref
  sqlSession.setConfigLocation("classpath:/.../..xml");    // value
  sqlSession.setMapperLocations("classpath*:/.../..xml");  // value

5. Bean л“ұлЎқ = м„Өкі„лҸ„ м ңм¶ң¶

Bean л“ұлЎқмқҖ "мқҙл ҮкІҢ л§Ңл“Өм–ҙмӨҳ"лқјлҠ” м„Өкі„лҸ„лҘј Springм—җ м ңм¶ңн•ҳлҠ” кІғмқҙлӢӨ.

<bean id="sqlSession" class="SqlSessionFactoryBean">
    <property name="dataSource" ref="dataSource" />
</bean>

мқҙ XMLмқҖ мҪ”л“ңк°Җ м•„лӢҲлӢӨ.
мқҙкұҙ Springм—җкІҢ ліҙлӮҙлҠ” "мЈјл¬ём„ң"лӢӨ.

мЈјл¬ём„ң лӮҙмҡ©:
  - мқҙлҰ„: sqlSession
  - мў…лҘҳ: SqlSessionFactoryBean
  - мһ¬лЈҢ: dataSourceлқјлҠ” Beanмқ„ л„Јм–ҙмӨҳ

Springмқҙ мқҙ мЈјл¬ём„ңлҘј мқҪкі :
  1. SqlSessionFactoryBean нҒҙлһҳмҠӨлҘј м°ҫлҠ”лӢӨ
  2. new SqlSessionFactoryBean()мңјлЎң к°қмІҙлҘј л§Ңл“ лӢӨ
  3. dataSource Beanмқ„ м°ҫм•„м„ң setDataSource()лЎң л„Јм–ҙмӨҖлӢӨ
  4. м»Ён…Ңмқҙл„Ҳм—җ "sqlSession"мқҙлқјлҠ” мқҙлҰ„мңјлЎң ліҙкҙҖн•ңлӢӨ
  5. лҲ„кө°к°Җ "sqlSession мӨҳ" н•ҳл©ҙ мқҙкұё кәјлӮҙмӨҖлӢӨ

6. Spring м»Ён…Ңмқҙл„Ҳ лӮҙл¶Җ¶

6.1 м»Ён…Ңмқҙл„Ҳ = Bean м ҖмһҘмҶҢ¶

Spring м»Ён…Ңмқҙл„Ҳ

мқҙлҰ„(id) к°қмІҙ(Bean)
"dataSource" BasicDataSource к°қмІҙ
"sqlSession" SqlSessionFactoryBean к°қмІҙ
"userService" UserServiceImpl к°қмІҙ
"lobHandler" DefaultLobHandler к°қмІҙ
... ...

= мқҙлҰ„(key) вҶ’ к°қмІҙ(value)мқҳ Mapкіј л№„мҠ·н•ҳлӢӨ = лӮҙл¶Җм ҒмңјлЎң Map<String, Object> к°ҷмқҖ кө¬мЎ°

6.2 Beanмқ„ к°Җм ёмҳӨлҠ” л°©лІ•¶

// л°©лІ• 1: мқҙлҰ„мңјлЎң к°Җм ёмҳӨкё°
ApplicationContext context = ...;
UserService service = (UserService) context.getBean("userService");

// л°©лІ• 2: нғҖмһ…мңјлЎң к°Җм ёмҳӨкё°
UserService service = context.getBean(UserService.class);

// л°©лІ• 3: м–ҙл…ён…Ңмқҙм…ҳмңјлЎң мЈјмһ…л°ӣкё° (к°ҖмһҘ мқјл°ҳм Ғ)
@Resource(name = "userService")
private UserService userService;
// вҶ’ Springмқҙ м»Ён…Ңмқҙл„Ҳм—җм„ң "userService" Beanмқ„ м°ҫм•„м„ң л„Јм–ҙмӨҖлӢӨ

7. мһҗмЈј н•ҳлҠ” мӢӨмҲҳ¶

7.1 Bean л“ұлЎқ м•Ҳ н•ҳкі  м“°кё°¶

// Bean л“ұлЎқ м•Ҳ н•ң нҒҙлһҳмҠӨлҘј @ResourceлЎң мЈјмһ…л°ӣмңјл©ҙ
@Resource
private MyNewService myNewService;  // вҶ’ м—җлҹ¬! Beanмқҙ м—ҶлӢӨ!

// м—җлҹ¬ л©”мӢңм§Җ:
// No qualifying bean of type 'MyNewService'
// = "MyNewServiceлқјлҠ” Beanмқҙ м»Ён…Ңмқҙл„Ҳм—җ м—ҶлӢӨ"

// н•ҙкІ°: XMLм—җ л“ұлЎқн•ҳкұ°лӮҳ, @Component м–ҙл…ён…Ңмқҙм…ҳ л¶ҷмқҙкё°

7.2 id мҳӨнғҖ¶

<bean id="dataSoruce" class="...BasicDataSource" />
<!--        вҶ‘ мҳӨнғҖ! dataSourceк°Җ м•„лӢҲлқј dataSoruce -->

<bean id="sqlSession" class="...SqlSessionFactoryBean">
    <property name="dataSource" ref="dataSource" />
    <!--                              вҶ‘ "dataSource" Beanмқ„ м°ҫлҠ”лҚ° м—ҶмқҢ! -->

</bean>

// м—җлҹ¬: No bean named 'dataSource' available

8. н•өмӢ¬ м •лҰ¬¶

н•өмӢ¬ м •лҰ¬

Bean = Spring м»Ён…Ңмқҙл„Ҳк°Җ мғқм„ұн•ҳкі  кҙҖлҰ¬н•ҳлҠ” к°қмІҙ

Bean л“ұлЎқ (XML): 

<bean id="мқҙлҰ„" class="нҒҙлһҳмҠӨ">
    <property name="н•„л“ң" value="к°’" />
    <property name="н•„л“ң" ref="лӢӨлҘёBeanмқҙлҰ„" />
</bean>

  • id = Beanмқҳ мқҙлҰ„ (кі мң  мӢқлі„мһҗ)
  • class = л§Ңл“Ө нҒҙлһҳмҠӨ (FQCN)
  • property name = setter л©”м„ңл“ңмҷҖ л§Өн•‘
  • value = л¬ёмһҗм—ҙ/мҲ«мһҗ к°’
  • ref = лӢӨлҘё Bean м°ёмЎ°

Spring м»Ён…Ңмқҙл„Ҳ = Beanл“Өмқҳ м ҖмһҘмҶҢ = Map<мқҙлҰ„, к°қмІҙ> к°ҷмқҖ кө¬мЎ°

лӢӨмқҢ мһҘ: DI (мқҳмЎҙм„ұ мЈјмһ…) вҶ’ "мҷң refлЎң лӢӨлҘё Beanмқ„ л„ЈлҠ” кұҙм§Җ" к№Ҡмқҙ нҢҢкі л“ лӢӨ
