05. Webhook vs API нҳём¶ң - м§„м§ң м°Ёмқҙ - Gamma¶

мқҙ мұ•н„°к°Җ м ңмқј мӨ‘мҡ”н•ҙ

мӢӨм ң л©ҳнҶ л§Ғм—җм„ң 12лІҲ л„ҳкІҢ к°ҷмқҖ м§Ҳл¬ёмқҙ л°ҳліөлҗҗм–ҙ.

"мқҙкІғлҸ„ API нҳём¶ңмқҙмһ–м•„" "мҷң мқҙкұҙ Webhookмқҙ м•„лӢҢлҚ°" "лӢӨ к°ҷмқҖ кұ° м•„лӢҲм•ј?"

м—¬кё°м„ң м „л¶Җ м •лҰ¬н•ңлӢӨ. мқҙ мұ•н„° лӢӨ мқҪкі лҸ„ н—·к°ҲлҰ¬л©ҙ мІҳмқҢл¶Җн„° лӢӨмӢң мқҪм–ҙ.

1. лЁјм Җ мқём •н•ҳмһҗ - "кё°мҲ м ҒмңјлЎң к°ҷлӢӨ"¶

Webhookкіј API нҳём¶ңмқҖ кё°мҲ м ҒмңјлЎң к°ҷм•„. л‘ҳ лӢӨ HTTP мҡ”мІӯмқҙм•ј.

н•ӯлӘ© API нҳём¶ң Webhook
н”„лЎңнҶ мҪң HTTP HTTP
л©”м„ңл“ң POST, GET л“ұ POST
лҚ°мқҙн„° нҳ•мӢқ JSON JSON
нҶөмӢ  л°©мӢқ мҡ”мІӯ вҶ’ мқ‘лӢө мҡ”мІӯ вҶ’ мқ‘лӢө

м°ЁмқҙлҠ” кё°мҲ мқҙ м•„лӢҲлқј "нҢЁн„ҙ"мқҙм•ј. м–ҙл–Ө кө¬мЎ°лЎң м“°лҠҗлғҗк°Җ лӢӨлҘё кұ°м§Җ.

лҶҚкө¬лЎң м№ҳл©ҙ:

кө¬л¶„ м„ӨлӘ…
мҠӣ кіөмқ„ кіЁлҢҖм—җ л„ЈлҠ” н–үмң„ (= HTTP мҡ”мІӯ)
л Ҳмқҙм—… мҠӣмқҳ н•ң мў…лҘҳ (= API нҳём¶ң)
лҚ©нҒ¬ мҠӣмқҳ н•ң мў…лҘҳ (= Webhook)

лҚ©нҒ¬лҸ„ мҠӣмқҙм•ј. к·јлҚ° мқҙлҰ„мқҙ л”°лЎң мһҲм–ҙ. л°©мӢқмқҙ лӢӨлҘҙлӢҲк№Ң.

2. м§„м§ң м°Ёмқҙ - "URL л“ұлЎқ мӢңмҠӨн…ң"¶

н•өмӢ¬ н•ң мӨ„

н”Ңлһ«нҸјмқҙ м ңкіөн•ҳлҠ” URL л“ұлЎқ мӢңмҠӨн…ңмқ„ нҶөн•ҙ нҳём¶ңлҗҳл©ҙ Webhook.

к°ңл°ңмһҗк°Җ мҪ”л“ңм—җм„ң м§Ғм ‘ нҳём¶ңн•ҳл©ҙ API нҳём¶ң.

3. н—·к°ҲлҰ¬лҠ” м§Ҳл¬ё 7к°Җм§Җ - "м „л¶Җ лӢөн•ңлӢӨ"¶

Q1: "л‘ҳ лӢӨ URL м“°мһ–м•„. лӯҗк°Җ лӢ¬лқј?"¶

лӢө

URLмқ„ м–ҙл””м—җ м“°лҠҗлғҗк°Җ лӢ¬лқј.

Webhook API нҳём¶ң
Zoom м„Өм • нҺҳмқҙм§Җм—җ URL мһ…л Ҙ мҪ”л“ңм—җ URL мһ‘м„ұ
м„Өм • н•ң лІҲмқҙл©ҙ лҒқ мҪ”л“ңм—җ л°•нҳҖмһҲмқҢ

Q2: "NC вҶ’ LMSлҸ„ мқҙлІӨнҠё(к°•мқҳ мў…лЈҢ) л•Ңл¬ём—җ мһҗлҸҷмңјлЎң ліҙлӮҙлҠ” кұҙлҚ°, мҷң Webhook м•„лӢҲм•ј?"¶

лӢө

мһҗлҸҷмқҙлқјкі  лӢӨ Webhookмқҙ м•„лӢҲм•ј.

NC вҶ’ LMS: мҡ°лҰ¬ мҪ”л“ңм—җм„ң http.post(lmsUrl, data) м§Ғм ‘ м§¬ вҶ’ API нҳём¶ң

Zoom вҶ’ NC: Zoom м„Өм •м—җ URL л“ұлЎқ, Zoomмқҙ м•Ңм•„м„ң ліҙлғ„ вҶ’ Webhook

мһҗлҸҷнҷ”мқҳ мЈјмІҙк°Җ лӢӨлҘҙлӢӨ:

мһҗлҸҷнҷ” мЈјмІҙ л°©мӢқ
мҡ°лҰ¬ мҪ”л“ңк°Җ мһҗлҸҷнҷ” API нҳём¶ң
н”Ңлһ«нҸј мӢңмҠӨн…ңмқҙ мһҗлҸҷнҷ” Webhook

Q3: "ліҙлӮҙлҠ” мҪ”л“ң м•Ҳ м§ңлҸ„ лҗҳл©ҙ Webhookмқҙлқјл©°? LMSк°Җ NC к°•мқҳ мғқм„ұ нҳём¶ңн•  л•ҢлҸ„ NCлҠ” л°ӣкё°л§Ң н•ҳмһ–м•„?"¶

лӢө

м—¬кё°м„ң Webhookмқҙлғҗ м•„лӢҲлғҗлҠ” л°ӣлҠ” мӘҪмқҙ м•„лӢҲлқј ліҙлӮҙлҠ” мӘҪмқ„ лҙҗм•ј н•ҙ.

  • LMS вҶ’ NC к°•мқҳ мғқм„ұ: LMS к°ңл°ңмһҗк°Җ нҳём¶ң мҪ”л“ңлҘј м§Ғм ‘ м§¬ вҶ’ API нҳём¶ң
  • Zoom вҶ’ NC Webhook: Zoom н”Ңлһ«нҸјмқҳ Webhook мӢңмҠӨн…ңмқҙ нҳём¶ң вҶ’ Webhook

NC мһ…мһҘм—җм„ңлҠ” л‘ҳ лӢӨ л°ӣкё°л§Ң н•ҙ. к·јлҚ° ліҙлӮҙлҠ” мӘҪмқҳ л°©мӢқмқҙ лӢӨлҘё кұ°м•ј.

мҡ°лҰ¬лҠ” к·ёлҹ° кұ° м•Ҳ л§Ңл“Өкі  мҪ”л“ңм—җм„ң м§Ғм ‘ нҳём¶ңн•ҙ. к·ёлһҳм„ң API нҳём¶ң.

Q6: "мҶҢм…ң лЎңк·ёмқё(л„ӨмқҙлІ„ лЎңк·ёмқё)лҸ„ Webhookмқҙм•ј?"¶

лӢө

м•„лӢҲм•ј. мҶҢм…ң лЎңк·ёмқёмқҖ OAuthлқјлҠ” лӢӨлҘё нҢЁн„ҙмқҙм•ј.

кө¬л¶„ Webhook мҶҢм…ң лЎңк·ёмқё
нҶөмӢ  м„ңлІ„ вҶ’ м„ңлІ„ лёҢлқјмҡ°м Җ лҰ¬лӢӨмқҙл үнҠё
нҠёлҰ¬кұ° мӢңмҠӨн…ң мқҙлІӨнҠё мӮ¬мҡ©мһҗ нҒҙлҰӯ
лӘ©м Ғ мқҙлІӨнҠё м•ҢлҰј мқёмҰқ/мқёк°Җ

Q7: "мҠӨмјҖмӨ„л§Ғ API нҳём¶ңмқҖ?"¶

лӢө

мқјл°ҳ API нҳём¶ңмқҙм•ј. мҡ°лҰ¬к°Җ "л§Өмқј 9мӢңм—җ нҳём¶ңн•ҙлқј"кі  мҪ”л“ң м§  кұ°лӢҲк№Ң.

  • Webhook: мғҒлҢҖк°Җ мқҙлІӨнҠё к°җм§Җн•ҳкі  мғҒлҢҖ мӢңмҠӨн…ңмқҙ нҳём¶ң вҶ’ Webhook

5. мӢӨм „ нҢҗлі„ м—°мҠө¶

мғҒнҷ© URL л“ұлЎқ мӢңмҠӨн…ң мһҲм–ҙ? кІ°лЎ 
Zoom вҶ’ NC н•ҷмғқ мһ…мһҘ Zoom Marketplaceм—җ мһҲмқҢ Webhook

нҒ¬лЎ мһЎмңјлЎң API нҳём¶ң м—ҶмқҢ. мҠӨмјҖмӨ„лҹ¬к°Җ м§Ғм ‘ нҳём¶ң API нҳём¶ң

LMS вҶ’ NC к°•мқҳ мғқм„ұ м—ҶмқҢ. LMS мҪ”л“ңм—җм„ң м§Ғм ‘ нҳём¶ң API нҳём¶ң

6. мҷң мқҙлҰ„мқ„ л”°лЎң л¶ҷмҳҖмқ„к№Ң?¶

мғқк°Ғн•ҙлҙҗ

кё°мҲ м ҒмңјлЎң к°ҷмқҖлҚ° мҷң көімқҙ лӢӨлҘё мқҙлҰ„мқ„ л¶ҷмҳҖмқ„к№Ң?

м„Өкі„ нҢЁн„ҙмқҙ лӢӨлҘҙкё° л•Ңл¬ёмқҙм•ј.

н•ӯлӘ© мқјл°ҳ API нҳём¶ң Webhook
кІ°н•©лҸ„ лҶ’мқҢ (URLмқҙ мҪ”л“ңм—җ л°•нһҳ) лӮ®мқҢ (URLмқ„ л“ұлЎқ/н•ҙм ң к°ҖлҠҘ)
мң м—°м„ұ мҪ”л“ң мҲҳм •н•ҙм•ј URL ліҖкІҪ м„Өм •м—җм„ң URL ліҖкІҪ к°ҖлҠҘ
нҷ•мһҘм„ұ мғҲ лҢҖмғҒ м¶”к°Җ = мҪ”л“ң мҲҳм • мғҲ лҢҖмғҒ = URL л“ұлЎқл§Ң
мЈјмІҙ нҳём¶ңмһҗк°Җ м ңм–ҙ мқҙлІӨнҠё л°ңмғқмһҗк°Җ м ңм–ҙ

WebhookмқҖ мқҙлІӨнҠё кё°л°ҳ м•„нӮӨн…ҚмІҳм—җм„ң н•өмӢ¬ нҢЁн„ҙмқҙм•ј. л§ҲмқҙнҒ¬лЎңм„ңл№„мҠӨ, мӢӨмӢңк°„ м•ҢлҰј, мҷёл¶Җ м—°лҸҷм—җм„ң н‘ңмӨҖмңјлЎң м“°м—¬.

7. м •лҰ¬¶

н•ӯлӘ© API нҳём¶ң Webhook
URL мң„м№ҳ мҪ”л“ңм—җ мһ‘м„ұ м„Өм • нҺҳмқҙм§Җм—җ л“ұлЎқ
нҳём¶ң мҪ”л“ң лӮҙк°Җ м§¬ н”Ңлһ«нҸјмқҙ л§Ңл“  кұ°
нғҖмқҙл°Қ лӮҙк°Җ м •н•Ё мқҙлІӨнҠё л°ңмғқ мӢң мһҗлҸҷ
кё°мҲ  HTTP мҡ”мІӯ HTTP мҡ”мІӯ (к°ҷмқҢ)
кҙҖкі„ мғҒмң„ к°ңл…җ API нҳём¶ңмқҳ н•ң мў…лҘҳ

мқҙ мұ•н„°м—җм„ң л°ҳл“ңмӢң кё°м–өн•  кІғ

Webhookкіј API нҳём¶ңмқҖ кё°мҲ м ҒмңјлЎң к°ҷм•„. л‘ҳ лӢӨ HTTP мҡ”мІӯмқҙм•ј.

м°ЁмқҙлҠ” нҢЁн„ҙмқҙм•ј: URL л“ұлЎқ мӢңмҠӨн…ңмқ„ нҶөн•ң мһҗлҸҷ нҳём¶ңмқҙл©ҙ Webhook, мҪ”л“ңм—җм„ң м§Ғм ‘ нҳём¶ңн•ҳл©ҙ API нҳём¶ң.

н—·к°ҲлҰ¬л©ҙ л”ұ н•ҳлӮҳл§Ң л¬јм–ҙлҙҗ: "ліҙлӮҙлҠ” мӘҪм—җ URL л“ұлЎқн•ҳлҠ” м„Өм • нҺҳмқҙм§Җк°Җ мһҲм–ҙ?"

нҷ•мқё л¬ём ң (5л¬ём ң)¶

лӢӨмқҢ л¬ём ңлҘј н’Җм–ҙлҙҗ. лӢө лӘ» н•ҳл©ҙ мң„м—җм„ң лӢӨмӢң мқҪм–ҙ.

Q1. Webhookкіј API нҳём¶ңмқҳ кё°мҲ м Ғ м°ЁмқҙлҠ” лӯҗм•ј?

Q2. NCк°Җ к°•мқҳ мў…лЈҢ нӣ„ LMSм—җ м¶ңкІ° лҚ°мқҙн„°лҘј мһҗлҸҷ м „мҶЎн•ңлӢӨ. мқҙкұҙ Webhookмқҙм•ј API нҳём¶ңмқҙм•ј? мқҙмң лҠ”?

Q3. Webhookмқём§Җ API нҳём¶ңмқём§Җ нҢҗлі„н•ҳлҠ” м§Ҳл¬ё н•ң к°Җм§ҖлҠ”?

Q4. "WebhookмқҖ API нҳём¶ңмқҙлӢӨ" - л§һм•„ нӢҖл Ө?

Q5. GitHubм—җм„ң мҪ”л“ңлҘј Pushн•ҳл©ҙ Jenkinsк°Җ мһҗлҸҷ л№Ңл“ңлҗңлӢӨ. GitHub вҶ’ Jenkins нҶөмӢ мқҖ Webhookмқҙм•ј API нҳём¶ңмқҙм•ј? мқҙмң лҠ”?

м •лӢө ліҙкё°

A1. кё°мҲ м Ғ м°ЁмқҙлҠ” м—ҶлӢӨ. л‘ҳ лӢӨ HTTP мҡ”мІӯ. м°ЁмқҙлҠ” нҢЁн„ҙ(URL л“ұлЎқ мӢңмҠӨн…ң мң л¬ҙ)мқҙлӢӨ.

A2. API нҳём¶ң. NC мҪ”л“ңм—җм„ң LMS URLмқ„ м§Ғм ‘ нҳём¶ңн•ҳлҠ” кұ°м§Җ, URL л“ұлЎқ мӢңмҠӨн…ңмқ„ нҶөн•ң кІҢ м•„лӢҲлӢҲк№Ң.

A3. "ліҙлӮҙлҠ” мӘҪм—җ URL л“ұлЎқн•ҳлҠ” м„Өм • нҺҳмқҙм§Җк°Җ мһҲм–ҙ?"

A4. л§һлӢӨ. WebhookмқҖ API нҳём¶ңмқҳ н•ң мў…лҘҳ(н•ҳмң„ к°ңл…җ)лӢӨ. лҚ©нҒ¬к°Җ мҠӣмқҳ н•ң мў…лҘҳмқҙл“Ҝ.

A5. Webhook. GitHub Settingsм—җ URL л“ұлЎқн•ҳлҠ” Webhook м„Өм • нҺҳмқҙм§Җк°Җ мһҲкі , Push мқҙлІӨнҠё л°ңмғқ мӢң GitHubмқҙ л“ұлЎқлҗң URL(Jenkins)лЎң мһҗлҸҷ нҳём¶ңн•ҳлӢҲк№Ң.
