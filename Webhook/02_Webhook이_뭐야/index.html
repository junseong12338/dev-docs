02. Webhookмқҙ лӯҗм•ј - Alpha¶

1. м „нҷ”к°Җ мҳӨлҠ” кұ°м•ј - "мқҙкІҢ лӯҗм•ј?"¶

м§ҖлӮң мұ•н„°м—җм„ң API нҳём¶ңмқҖ "лӮҙк°Җ м „нҷ”н•ҳлҠ” кІғ"мқҙлқјкі  н–Ҳм§Җ?

WebhookмқҖ л°ҳлҢҖм•ј. "м „нҷ”к°Җ мҳӨлҠ” кІғ"мқҙм•ј.

мӢӨмғқнҷң л№„мң 

м№ҳнӮЁ мЈјл¬ён•  л•Ң лӮҙ м „нҷ”лІҲнҳёлҘј м•Ңл ӨмӨ¬м–ҙ.

  • м№ҳнӮЁ лӢӨ лҗҳл©ҙ м№ҳнӮЁм§‘мқҙ м•Ңм•„м„ң м „нҷ”н•ҙ
  • лӮҙк°Җ м „нҷ” м•Ҳ н•ҙлҸ„ м•Ңл ӨмӨҳ
  • лӢЁ, лӮҙ лІҲнҳёлҘј лҜёлҰ¬ м•Ңл ӨмӨ¬м–ҙм•ј к°ҖлҠҘн•ҙ
кө¬л¶„ м№ҳнӮЁ мЈјл¬ё (API нҳём¶ң) м№ҳнӮЁ мҷ„м„ұ м•ҢлҰј (Webhook)
лҲ„к°Җ м „нҷ” лӮҙк°Җ м№ҳнӮЁм§‘мқҙ
м–ём ң лӮҙк°Җ мӣҗн•  л•Ң м№ҳнӮЁ лӢӨ лҗҗмқ„ л•Ң
м „м ң мЎ°кұҙ м№ҳнӮЁм§‘ лІҲнҳё м•Ңкё° лӮҙ лІҲнҳё м•Ңл ӨмЈјкё°

л№„мң лҠ” м—¬кё°к№Ңм§Җ. м§„м§ң м •мқҳ к°„лӢӨ.

н•ң мӨ„ м •мқҳ

Webhook = мҷёл¶Җ м„ңл№„мҠӨм—җ лӮҙ URLмқ„ л“ұлЎқн•ҙл‘җл©ҙ, мқҙлІӨнҠё л°ңмғқ мӢң к·ё м„ңл№„мҠӨк°Җ лӮҙ URLлЎң мһҗлҸҷмңјлЎң HTTP мҡ”мІӯмқ„ ліҙлӮҙмЈјлҠ” кІғ

Hook к°Ҳкі лҰ¬, кұём–ҙл‘җлӢӨ

"мӣ№м—җ к°Ҳкі лҰ¬лҘј кұём–ҙл‘”лӢӨ" = мқҙлІӨнҠёк°Җ л°ңмғқн•ҳлҠ” кіім—җ лӮҙ URLмқ„ кұём–ҙл‘җл©ҙ, мқҙлІӨнҠёк°Җ м§ҖлӮҳк°Ҳ л•Ң к°Ҳкі лҰ¬м—җ кұёл Өм„ң лӮҳн•ңн…Ң м•Ңл ӨмӨҖлӢӨ.

н”„лЎңк·ёлһҳл°Қм—җм„ң "Hook"мқҖ "нҠ№м • мқҙлІӨнҠёк°Җ л°ңмғқн•  л•Ң мӢӨн–үлҗҳлҠ” мҪңл°ұ"мқҙлқјлҠ” лң»мңјлЎң м“°м—¬. Git Hook, React Hook лӢӨ к°ҷмқҖ л§ҘлқҪмқҙм•ј.

4. Webhookмқ„ м“°лҠ” м„ңл№„мҠӨл“Ө - "м–ҙл””м„ң м“°лҠ” кұҙлҚ°?"¶

WebhookмқҖ мҡ°лҰ¬ н”„лЎңм қнҠём—җм„ңл§Ң м“°лҠ” кІҢ м•„лӢҲм•ј. м—…кі„ н‘ңмӨҖмқҙм•ј.

м„ңл№„мҠӨ Webhook мқҙлІӨнҠё мҳҲмӢң мҡ©лҸ„
Zoom н•ҷмғқ мһ…мһҘ, нҮҙмһҘ, к°•мқҳ мў…лЈҢ м¶ңкІ° кҙҖлҰ¬ (мҡ°лҰ¬ н”„лЎңм қнҠё!)
GitHub Push, PR мғқм„ұ, Issue мғқм„ұ CI/CD мһҗлҸҷнҷ”
Stripe кІ°м ң мҷ„лЈҢ, нҷҳл¶Ҳ кІ°м ң мІҳлҰ¬
Slack л©”мӢңм§Җ мҲҳмӢ  лҙҮ мһҗлҸҷ мқ‘лӢө
м№ҙм№ҙмҳӨнҺҳмқҙ кІ°м ң мҠ№мқё, м·ЁмҶҢ кІ°м ң м—°лҸҷ

кіөнҶөм җ ліҙм—¬?

м „л¶Җ "мҷёл¶Җ н”Ңлһ«нҸјм—җм„ң мқҙлІӨнҠёк°Җ л°ңмғқн•ҳл©ҙ лӮҙ м„ңлІ„лЎң м•Ңл ӨмЈјлҠ” кІғ"мқҙм•ј.

мқҙ н”Ңлһ«нҸјл“ӨмқҖ м „л¶Җ Webhook м„Өм • нҺҳмқҙм§Җк°Җ мһҲм–ҙ. URL л“ұлЎқн•ҳлҠ” кіі.

5. мҷң Webhookмқ„ мҚЁм•ј н•ҙ? - "к·ёлғҘ API нҳём¶ңн•ҳл©ҙ м•Ҳ лҸј?"¶

Zoomм—җм„ң н•ҷмғқмқҙ мһ…мһҘн–ҲлҠ”м§Җ м•Ңкі  мӢ¶м–ҙ. л°©лІ•мқҙ л‘җ к°Җм§Җм•ј:

NC: "Zoomм•„ лҲ„кө¬ л“Өм–ҙмҷ”м–ҙ?" вҶ’ "м—Ҷм–ҙ"
NC: "Zoomм•„ лҲ„кө¬ л“Өм–ҙмҷ”м–ҙ?" вҶ’ "м—Ҷм–ҙ"

... л¬ҙн•ңл°ҳліө ...

л¬ём ңм җ:

  • л§ӨлІҲ л¬јм–ҙлҙҗм•ј н•ҙ (м„ңлІ„ л¶Җн•ҳ)
  • л¬јм–ҙліҙлҠ” мӮ¬мқҙм—җ мқҙлІӨнҠё лҶ“м№  мҲҳ мһҲм–ҙ (мӢӨмӢңк°„м„ұ л¶ҖмЎұ)
  • Zoomмқҙ "л„Ҳ л„Ҳл¬ҙ мһҗмЈј л¬јм–ҙлҙҗ, м°ЁлӢЁ!" (Rate Limit)
NC: (к°Җл§ҢнһҲ мһҲмқҢ)
Zoom: "н•ҷмғқA л“Өм–ҙмҷ”м–ҙ" (мҰүмӢң м•Ңл ӨмӨҢ)
NC: (к°Җл§ҢнһҲ мһҲмқҢ)
Zoom: "н•ҷмғқB л“Өм–ҙмҷ”м–ҙ" (мҰүмӢң м•Ңл ӨмӨҢ)

мһҘм җ:

  • л¬јм–ҙліј н•„мҡ” м—Ҷм–ҙ (м„ңлІ„ л¶Җн•ҳ м—ҶмқҢ)
  • мҰүмӢң м•Ңл ӨмӨҳ (мӢӨмӢңк°„)
  • Zoomмқҙ м°ЁлӢЁн•  мқҙмң  м—Ҷм–ҙ

мӢӨмӢңк°„м„ұ лӮ®мқҢ (мЈјкё°м—җ л”°лқј) лҶ’мқҢ (мҰүмӢң)

Rate Limit мң„н—ҳ м•Ҳм „

6. м •лҰ¬¶

н•ӯлӘ© лӮҙмҡ©
WebhookмқҙлһҖ мҷёл¶Җ м„ңл№„мҠӨм—җ URL л“ұлЎқ вҶ’ мқҙлІӨнҠё мӢң мһҗлҸҷ HTTP мҡ”мІӯ
мқҙлҰ„ лң» Web + Hook = мқҙлІӨнҠём—җ к°Ҳкі лҰ¬ кұёкё°
мҷң м“°лғҗ мҷёл¶Җ м„ңл№„мҠӨ лӮҙл¶Җ мқҙлІӨнҠёлҘј м•Ң л°©лІ•мқҙ мқҙкІғлҝҗ
Pollingкіј м°Ёмқҙ Polling=лӮҙк°Җ л¬јм–ҙлҙ„, Webhook=мғҒлҢҖк°Җ м•Ңл ӨмӨҢ
мҡ°лҰ¬к°Җ м§ңлҠ” мҪ”л“ң л°ӣлҠ” мҪ”л“ңл§Ң м§ лӢӨ

мқҙ мұ•н„°м—җм„ң л°ҳл“ңмӢң кё°м–өн•  кІғ

Webhook = мҷёл¶Җ м„ңл№„мҠӨк°Җ мқҙлІӨнҠё л°ңмғқ мӢң лӮҙ URLлЎң м•Ңм•„м„ң м•Ңл ӨмЈјлҠ” кІғ.

ліҙлӮҙлҠ” мҪ”л“ңлҠ” мғҒлҢҖк°Җ л§Ңл“  кұ°. мҡ°лҰ¬лҠ” л°ӣлҠ” мҪ”л“ңл§Ң м§ лӢӨ.

м •лӢө ліҙкё°

A1. к°Ҳкі лҰ¬. мқҙлІӨнҠёк°Җ л°ңмғқн•ҳлҠ” кіім—җ лӮҙ URLмқ„ кұём–ҙл‘җлҠ” кІғ. мқҙлІӨнҠёк°Җ м§ҖлӮҳк°Ҳ л•Ң к°Ҳкі лҰ¬м—җ кұёл Өм„ң м•ҢлҰјмқҙ мҳЁлӢӨ.

A2. мҷёл¶Җ м„ңл№„мҠӨ(Zoom л“ұ)м—җ лӮҙ м„ңлІ„ URLмқ„ л“ұлЎқн•ҙм•ј н•ңлӢӨ.

A3. (1) мӢӨмӢңк°„м„ұ - мҰүмӢң м•Ңл ӨмӨҢ (2) м„ңлІ„ л¶Җн•ҳ к°җмҶҢ - л¶Ҳн•„мҡ”н•ң мҡ”мІӯ м—ҶмқҢ (3) Rate Limit м•Ҳм „ - л°ҳліө нҳём¶ң м•Ҳ н•ҙлҸ„ лҗЁ. + мҷёл¶Җ м„ңлІ„м—җ мҡ°лҰ¬ мҪ”л“ңлҘј мӢ¬мқ„ мҲҳ м—ҶмңјлӢҲк№Ң.

A4. Zoom к°ңл°ңмһҗ. Webhookм—җм„ң ліҙлӮҙлҠ” мҪ”л“ңлҠ” н”Ңлһ«нҸј(Zoom)мқҙ л§Ңл“  кІғмқҙкі , мҡ°лҰ¬лҠ” л°ӣлҠ” мҪ”л“ңл§Ң м§ лӢӨ.
