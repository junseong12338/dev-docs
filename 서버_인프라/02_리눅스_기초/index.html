02. лҰ¬лҲ…мҠӨ кё°мҙҲ¶

"лҰ¬лҲ…мҠӨ лӘ…л №м–ҙ мҷёмӣҢм•ј н•ҙмҡ”?" вҶ’ мҷёмҡ°лҠ” кІҢ м•„лӢҲлқј мқҙн•ҙн•ҳлҠ” кұ°м•ј. лӘ…л №м–ҙк°Җ мҷң к·ёл ҮкІҢ мғқкІјлҠ”м§Җ м•Ңл©ҙ мҷёмҡё н•„мҡ” м—Ҷм–ҙ.

рҹҹў мҷң м„ңлІ„лҠ” лҰ¬лҲ…мҠӨмқёк°Җ?¶

л№„көҗ Windows Linux
лқјмқҙм„ мҠӨ мң лЈҢ л¬ҙлЈҢ (лҢҖл¶Җл¶„)
GUI мһҲмқҢ (л§Ҳмҡ°мҠӨ нҒҙлҰӯ) м—ҶмқҢ (н„°лҜёл„җ лӘ…л №м–ҙ)
м•Ҳм •м„ұ мһ¬л¶ҖнҢ… мһҗмЈј н•„мҡ” мҲҳл…„к°„ мһ¬л¶ҖнҢ… м—Ҷмқҙ мҡҙмҳҒ к°ҖлҠҘ
мһҗмӣҗ мӮ¬мҡ© GUI л•Ңл¬ём—җ л©”лӘЁлҰ¬ л§Һмқҙ м”Җ GUI м—Ҷм–ҙм„ң к°ҖлІјмӣҖ
м„ңлІ„ м җмң мңЁ ~20% ~80%

кІ°лЎ : м„ңлІ„лҠ” лӘЁлӢҲн„° м—Ҷмқҙ 24мӢңк°„ лҸҢм•„к°Җм•ј н•ҳлӢҲк№Ң, к°ҖліҚкі  м•Ҳм •м Ғмқё лҰ¬лҲ…мҠӨк°Җ лӢө.

рҹҹў л””л үнҶ лҰ¬ кө¬мЎ° (мқҙкұ° лӘЁлҘҙл©ҙ м„ңлІ„м—җм„ң кёё мһғмқҢ)¶

WindowsлҠ” C:\, D:\ л“ңлқјмқҙлёҢ. лҰ¬лҲ…мҠӨлҠ” м „л¶Җ / (лЈЁнҠё)м—җм„ң мӢңмһ‘н•ңлӢӨ.

мӢӨл¬ҙм—җм„ң мһҗмЈј к°ҖлҠ” кІҪлЎң¶

кІҪлЎң мҡ©лҸ„ мҷёмҡ°лҠ” лІ•
/etc/ м„Өм • нҢҢмқј et cetera (кё°нғҖ л“ұл“ұ)
/var/log/ лЎңк·ё нҢҢмқј variable (ліҖн•ҳлҠ” лҚ°мқҙн„°)
/tmp/ мһ„мӢң нҢҢмқј temporary
/home/ мӮ¬мҡ©мһҗ нҷҲ к·ёлғҘ home
/root/ кҙҖлҰ¬мһҗ нҷҲ root мӮ¬мҡ©мһҗмқҳ home

рҹҹў н•„мҲҳ лӘ…л №м–ҙ (мқҙкІғл§Ң м•Ңл©ҙ м„ңлІ„м—җм„ң м•Ҳ мЈҪмқҢ)¶

tar лӘ…л №м–ҙ мҳөм…ҳ мҷёмҡ°лҠ” лІ•¶

tar [мҳөм…ҳ] [нҢҢмқјлӘ…] [лҢҖмғҒ]

c = create (л§Ңл“Өкё°)
x = extract (н’Җкё°)
z = gzip (gz м••м¶•)
f = file (нҢҢмқј м§Җм •)
v = verbose (кіјм • н‘ңмӢң)

#   x = н’Җкё°
#   z = gzip
#   f = нҢҢмқјлӘ…

#   t = лӘ©лЎқ ліҙкё°

# "Removing leading '/'" кІҪкі
