## 핵심 원칙

```
1. 용어를 '안다'는 건 남에게 설명할 수 있다는 뜻이다
2. 숫자를 '읽는다'는 건 정상/비정상을 판단할 수 있다는 뜻이다
3. 리포트를 '분석한다'는 건 원인→결과→조치를 도출할 수 있다는 뜻이다
```
