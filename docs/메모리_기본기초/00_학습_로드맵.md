## 학습 원칙

1. **순서대로** 읽을 것. 뛰어넘지 말 것.
2. 각 챕터 끝 **확인 문제** 풀 것. 틀리면 다시 읽을 것.
3. 코드 예시는 **직접 타이핑** 할 것. 복붙 금지.
4. 최종시험 **80점 이상**이면 합격.
