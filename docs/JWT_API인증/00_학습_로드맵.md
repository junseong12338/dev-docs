## 학습 원칙

1. **순서대로** 읽을 것. 뛰어넘지 말 것.
2. 각 챕터 끝 **확인 문제** 풀 것. 틀리면 다시 읽을 것.
3. 코드 예시는 **직접 타이핑** 할 것. 복붙 금지.
4. 최종시험 **80점 이상**이면 합격.

## 왜 이 순서야?

```
JWT가 뭔지 모르는데 구조를 볼 수 있어?          → 01 먼저
암호화/인코딩 구분 못 하는데 서명을 이해해?      → 02 먼저
구조 모르는데 생성/검증 흐름을 알아?             → 03 먼저
생성/검증 모르는데 API 인증을 이해해?            → 04 먼저
JWT만 알고 AES는 모르는데 LMS 연동을 해?        → 06 먼저
보안 고려 없이 실전 코드를 짜?                   → 07 먼저
전부 모르면서 시험을 봐?                         → 01~08 다 읽고 와
```

각 단계가 다음 단계의 **선행 지식**이야. 뛰어넘으면 무조건 삽질한다.
