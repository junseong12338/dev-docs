### Q30. [실전 종합] (4점)

```java
public void syncStudents(String subjectNo) throws Exception {
    String response = callGnuApi("/api/hs/student?subjectNo=" + subjectNo);
    JSONObject root = (JSONObject) new JSONParser().parse(response);
    JSONArray studentList = (JSONArray) ((JSONObject) root.get("data")).get("studentList");

    // 기존 수강생 전체 삭제
    enrollmentDAO.deleteAllBySubject(subjectNo);

    // 새로 INSERT
    for (int i = 0; i < studentList.size(); i++) {
        JSONObject student = (JSONObject) studentList.get(i);
        String studentNo = (String) student.get("studentNo");
        enrollmentDAO.insertEnrollment(subjectNo, studentNo);
    }
}
```
