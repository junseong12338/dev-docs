05. мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠё - Gamma¶

1. мқҙкІҢ лӯҗм•ј? вҖ” "мһҘл°”кө¬лӢҲ"¶

мҳЁлқјмқё мҮјн•‘н•  л•Ң л¬јкұҙмқ„ л°”лЎң кІ°м ң м•Ҳ н•ҳмһ–м•„. мһҘл°”кө¬лӢҲм—җ лӢҙм•„л’ҖлӢӨк°Җ н•ң лІҲм—җ кІ°м ңн•ҳм§Җ.

JPAлҸ„ л§Ҳм°¬к°Җм§Җ. м—”нӢ°нӢ°лҘј DBм—җ л°”лЎң м•Ҳ л„Јм–ҙ. мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠёлқјлҠ” мһҘл°”кө¬лӢҲм—җ лӢҙм•„л’ҖлӢӨк°Җ, нҠёлһңмһӯм…ҳ лҒқлӮ  л•Ң н•ң лІҲм—җ DBм—җ л°ҳмҳҒн•ҙ.

мқҙкІҢ JPAмқҳ н•өмӢ¬ к°ңл…җмқҙм•ј. мқҙкұ° лӘЁлҘҙл©ҙ JPAлҘј м“°лҠ” кІҢ м•„лӢҲлқј JPAн•ңн…Ң лҒҢл ӨлӢӨлӢҲлҠ” кұ°м•ј.

2. м–ҙл–»кІҢ лҸҢм•„к°Җ?¶

м—”нӢ°нӢ° мғқлӘ…мЈјкё° 4лӢЁкі„¶

stateDiagram-v2
    [*] --> л№„мҳҒмҶҚ: new Term()
    л№„мҳҒмҶҚ --> мҳҒмҶҚ: save()
    мҳҒмҶҚ --> мӨҖмҳҒмҶҚ: нҠёлһңмһӯм…ҳ лҒқ
    мҳҒмҶҚ --> мӮӯм ң: delete()

    л№„мҳҒмҶҚ: (new)
    л№„мҳҒмҶҚ: к·ёлғҘ мһҗл°” к°қмІҙ

    мҳҒмҶҚ: (managed)
    мҳҒмҶҚ: мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠёк°Җ кҙҖлҰ¬н•ҳлҠ” мғҒнғң

    мӨҖмҳҒмҶҚ: (detached)
    мӨҖмҳҒмҶҚ: кҙҖлҰ¬м—җм„ң л№ м§җ (нҠёлһңмһӯм…ҳ л°–)

    мӮӯм ң: (removed)
мғҒнғң м„ӨлӘ… мҳҲмӢң
л№„мҳҒмҶҚ newлЎң л§Ңл“  мғҒнғң. JPAк°Җ лӘЁлҰ„ Term term = new Term()
мҳҒмҶҚ save() нӣ„. JPAк°Җ кҙҖлҰ¬ termRepository.save(term)
мӨҖмҳҒмҶҚ нҠёлһңмһӯм…ҳ лҒқлӮЁ. JPAк°Җ лҚ” мқҙмғҒ м•Ҳ лҙ„ л©”м„ңл“ң лҰ¬н„ҙ нӣ„
мӮӯм ң delete() нҳём¶ң. DBм—җм„ң м§Җмҡё мҳҲм • termRepository.delete(term)

1м°Ё мәҗмӢң¶

к°ҷмқҖ нҠёлһңмһӯм…ҳ м•Ҳм—җм„ң к°ҷмқҖ PKлЎң 2лІҲ мЎ°нҡҢн•ҳл©ҙ?

@Transactional
public void example() {
    Term term1 = termRepository.findById("2026-1").orElseThrow();  // SQL мӢӨн–ү O
    Term term2 = termRepository.findById("2026-1").orElseThrow();  // SQL мӢӨн–ү X (мәҗмӢң!)

    System.out.println(term1 == term2);  // true вҖ” к°ҷмқҖ к°қмІҙ!
}

JPA: SQL 1лІҲл§Ң мӢӨн–ү. л‘җ лІҲм§ёлҠ” 1м°Ё мәҗмӢңм—җм„ң к°Җм ёмҳҙ. MyBatis: SQL 2лІҲ мӢӨн–ү. мәҗмӢң м—ҶмқҢ (2м°Ё мәҗмӢң м„Өм • м•Ҳ н–Ҳмңјл©ҙ).

ліҖкІҪ к°җм§Җ (Dirty Checking) вҖ” мқҙкІҢ н•өмӢ¬¶

@Transactional
public void updateTermName(String termCd, String newName) {
    Term term = termRepository.findById(termCd).orElseThrow();  // мҳҒмҶҚ мғҒнғң
    term.setTermName(newName);  // н•„л“ң к°’л§Ң л°”кҝҲ
    // save() м•Ҳ л¶Ҳлҹ¬лҸ„ лҗЁ! нҠёлһңмһӯм…ҳ лҒқлӮ  л•Ң JPAк°Җ м•Ңм•„м„ң UPDATE
}

JPAк°Җ нҠёлһңмһӯм…ҳ лҒқлӮ  л•Ң "м•„к№Ң мЎ°нҡҢн•ң кұ°лһ‘ м§ҖкёҲ к°’мқҙ лӢӨлҘҙл„Ө?" вҶ’ мһҗлҸҷ UPDATE SQL мғқм„ұ.

MyBatisмҳҖмңјл©ҙ: 

public void updateTermName(String termCd, String newName) {
    TermVO vo = termMapper.selectTerm(termCd);
    vo.setTermName(newName);
    termMapper.updateTerm(vo);  // л°ҳл“ңмӢң нҳём¶ңн•ҙм•ј DB л°ҳмҳҒ!
}

JPA: setл§Ң н•ҳл©ҙ лҒқ. MyBatis: set н•ң л’Ө л°ҳл“ңмӢң update л©”м„ңл“ң нҳём¶ң.

NexClass мӢӨм „ вҖ” LessonService.updateLesson()¶

@Transactional
public Lesson updateLesson(String lessonCd, Lesson updateData) {
    Lesson lesson = getLesson(lessonCd);                    // мҳҒмҶҚ мғҒнғң
    if (updateData.getLessonName() != null) {
        lesson.setLessonName(updateData.getLessonName());   // к°’л§Ң л°”кҝҲ
    }
    // ...
    return lessonRepository.save(lesson);  // мӮ¬мӢӨ мқҙ save() м—Ҷм–ҙлҸ„ лҗЁ
}

@Transactional м•Ҳмқҙлқј save() м—Ҷм–ҙлҸ„ ліҖкІҪ к°җм§ҖлЎң UPDATE лҗЁ. н•ҳм§Җл§Ң NexClassм—җм„ңлҠ” лӘ…мӢңм ҒмңјлЎң save() нҳём¶ңн•ҙм„ң к°ҖлҸ…м„ұмқ„ лҶ’мқё кұ°м•ј.

3. flush вҖ” "мһҘл°”кө¬лӢҲ кІ°м ң"¶

мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠёмқҳ ліҖкІҪ лӮҙмҡ©мқ„ DBм—җ л°Җм–ҙлӮҙлҠ” кІғ.

нҸүмҶҢ: мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠём—җ мҢ“м•„л‘ 
flush мӢңм җ: DBм—җ SQL лӮ лҰј
  вҶ’ нҠёлһңмһӯм…ҳ м»Өл°Ӣ м§Ғм „ (мһҗлҸҷ)
  вҶ’ JPQL мҝјлҰ¬ мӢӨн–ү м§Ғм „ (мһҗлҸҷ)
  вҶ’ entityManager.flush() м§Ғм ‘ нҳём¶ң (мҲҳлҸҷ)

ліҙнҶө мӢ кІҪ м•Ҳ мҚЁлҸ„ лҸј. нҠёлһңмһӯм…ҳ лҒқлӮҳл©ҙ мһҗлҸҷ flush.

4. мЈјмқҳмӮ¬н•ӯ / н•Ём •¶

н•Ём • 1: @Transactional м—Ҷмқҙ ліҖкІҪ к°җм§Җ кё°лҢҖ @Transactional м—Ҷмңјл©ҙ мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠё м•Ҳ л§Ңл“Өм–ҙм ё. setл§Ң н•ҳкі  save() м•Ҳ л¶ҖлҘҙл©ҙ DB л°ҳмҳҒ м•Ҳ лҗЁ.

н•Ём • 2: нҠёлһңмһӯм…ҳ л°–м—җм„ң м§Җм—° лЎңл”© м—°кҙҖкҙҖкі„ м„Өм •н•ң м—”нӢ°нӢ°лҘј нҠёлһңмһӯм…ҳ л°–м—җм„ң м ‘к·јн•ҳл©ҙ LazyInitializationException н„°м ё.

н•Ём • 3: MyBatis мҠөкҙҖмңјлЎң л§ӨлІҲ save() ліҖкІҪ к°җм§Җк°Җ лҗҳлҠ” @Transactional м•Ҳм—җм„ң л§ӨлІҲ save() л¶ҖлҘҙлҠ” кұҙ л¶Ҳн•„мҡ”н•ң мҪ”л“ң. нӢҖлҰ° кұҙ м•„лӢҢлҚ° JPAлӢөм§Җ м•Ҡм•„.

5. м •лҰ¬¶

к°ңл…җ MyBatis JPA
к°ҷмқҖ PK 2лІҲ мЎ°нҡҢ SQL 2лІҲ SQL 1лІҲ (1м°Ё мәҗмӢң)
к°’ ліҖкІҪ нӣ„ DB л°ҳмҳҒ mapper.update() н•„мҲҳ setл§Ң н•ҳл©ҙ мһҗлҸҷ (ліҖкІҪ к°җм§Җ)
нҠёлһңмһӯм…ҳ SqlSession мҲҳлҸҷ кҙҖлҰ¬ @Transactional м–ҙл…ён…Ңмқҙм…ҳ
мһҘл°”кө¬лӢҲ к°ңл…җ м—ҶмқҢ мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠё

мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠёлҠ” JPAмқҳ мһҘл°”кө¬лӢҲ. м—”нӢ°нӢ°лҘј кҙҖлҰ¬н•ҳлӢӨк°Җ нҠёлһңмһӯм…ҳ лҒқлӮ  л•Ң DBм—җ л°ҳмҳҒн•ңлӢӨ. ліҖкІҪ к°җм§Җ лҚ•л¶„м—җ setл§Ң н•ҙлҸ„ UPDATEк°Җ мһҗлҸҷмңјлЎң лӮҳк°„лӢӨ.

нҷ•мқё л¬ём ң¶

Q1. мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠёмқҳ 1м°Ё мәҗмӢңк°Җ н•ҳлҠ” мқјмқҖ?

Q2. ліҖкІҪ к°җм§Җ(Dirty Checking)к°Җ лӯҗм•ј? MyBatisмҷҖ л№„көҗн•ҙм„ң.

Q3. @Transactional м—Ҷмқҙ entity.setName("мғҲмқҙлҰ„") н•ҳл©ҙ DBм—җ л°ҳмҳҒлҸј?

Q4. NexClass LessonService.updateLesson()м—җм„ң save()лҘј л№јлҸ„ лҗҳлҠ” мқҙмң лҠ”?

м •лӢө ліҙкё°

A1. к°ҷмқҖ нҠёлһңмһӯм…ҳ м•Ҳм—җм„ң к°ҷмқҖ PKлЎң мЎ°нҡҢн•ҳл©ҙ DB лҢҖмӢ  мәҗмӢңм—җм„ң л°ҳнҷҳ. SQL мӢӨн–ү нҡҹмҲҳлҘј мӨ„м—¬мӨҢ.

A2. @Transactional м•Ҳм—җм„ң мҳҒмҶҚ м—”нӢ°нӢ°мқҳ н•„л“ңлҘј л°”кҫёл©ҙ, нҠёлһңмһӯм…ҳ лҒқлӮ  л•Ң JPAк°Җ мӣҗліёкіј л№„көҗн•ҙм„ң мһҗлҸҷ UPDATE. MyBatisлҠ” л°ҳл“ңмӢң mapper.update()лҘј м§Ғм ‘ нҳём¶ңн•ҙм•ј н•Ё.

A3. м•Ҳ лҗЁ. @Transactional м—Ҷмңјл©ҙ мҳҒмҶҚм„ұ м»Ён…ҚмҠӨнҠёк°Җ кҙҖлҰ¬ м•Ҳ н•ҙм„ң ліҖкІҪ к°җм§Җк°Җ мһ‘лҸҷ м•Ҳ н•Ё. save()лҘј лӘ…мӢңм ҒмңјлЎң нҳём¶ңн•ҙм•ј н•Ё.

A4. @Transactional м•Ҳмқҙлқј ліҖкІҪ к°җм§Җк°Җ мһ‘лҸҷ. нҠёлһңмһӯм…ҳ лҒқлӮ  л•Ң JPAк°Җ ліҖкІҪлҗң н•„л“ңлҘј к°җм§Җн•ҙм„ң мһҗлҸҷ UPDATE. save()лҠ” к°ҖлҸ…м„ұмқ„ мң„н•ҙ лӘ…мӢңм ҒмңјлЎң м“ҙ кІғ.
