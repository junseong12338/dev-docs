11. 최종시험 - Black Hole¶

Black Hole

"There are no two words in the English language more harmful than 'good job'."

01~10 전 범위. 30문제. 100점 만점. 여기서 80점 못 넘기면 처음부터 다시 읽어.

시험 규칙¶

항목 규칙
합격 기준 80점 이상
시간 제한 60분
오픈북 금지. 머리에 없으면 모르는 거야.
검색 금지. 구글 켜는 순간 불합격.
부분 점수 서술형만 부분 점수 있음
총점 100점

배점 구성¶

유형 문항 수 문항당 점수 합계 비고
객관식 (4지선다) 10문제 2점 20점 개념 확인
단답형 8문제 3점 24점 정확한 이해
코드/명령어 분석 7문제 5점 35점 실전 능력
서술형 5문제 4~5점 21점 깊이 있는 이해
합계 30문제 100점

시작하기 전에.

답을 "아마 이거일 걸?"로 쓰지 마. 근거 없는 답은 답이 아니야. 모르면 모른다고 써. 찍어서 맞히는 건 실력이 아니야.

(A) 100 바이트 (B) 1,000 바이트 (C) 1,024 바이트 (D) 1,048 바이트

Q2. [01장] (2점)¶

메모리 계층 구조에서 CPU에 가장 가까운 것부터 먼 것 순서로 올바르게 나열한 것은?

(A) RAM → 레지스터 → 캐시 → 디스크 (B) 레지스터 → 캐시 → RAM → 디스크 (C) 캐시 → 레지스터 → RAM → 디스크 (D) 레지스터 → RAM → 캐시 → 디스크

Linux에서 ps -ef 명령어의 PID 컬럼이 1인 프로세스는 무엇인가?

(A) 커널 자체 (B) init/systemd (최초 사용자 프로세스) (C) bash 쉘 (D) 가장 먼저 실행된 Java 프로세스

Q4. [03장] (2점)¶

free -h 출력에서 available이 의미하는 것은?
