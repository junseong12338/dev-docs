18. CORSмҷҖ Same-Origin Policy¶

мқҙ мұ•н„°мқҳ н•өмӢ¬

лёҢлқјмҡ°м ҖлҠ” лӢӨлҘё м¶ңмІҳ(Origin)мқҳ лҰ¬мҶҢмҠӨ м ‘к·јмқ„ кё°ліём ҒмңјлЎң л§үлҠ”лӢӨ. мқҙкұё Same-Origin Policyлқјкі  н•ҙ. к·ёлҰ¬кі  "мқҙ OriginмқҖ кҙңм°®м•„, н—Ҳмҡ©н•ҙ"лқјкі  м„ңлІ„к°Җ лӘ…мӢңм ҒмңјлЎң м—ҙм–ҙмЈјлҠ” кІҢ CORSлӢӨ. н”„лЎ нҠём—”л“ңмҷҖ л°ұм—”л“ңлҘј л¶„лҰ¬н•ҙм„ң к°ңл°ңн•ҳл©ҙ 100% л§ҢлӮҳлҠ” мЈјм ңм•ј. мқҙкұ° лӘЁлҘҙл©ҙ мҪҳмҶ”м—җ л№Ёк°„ CORS м—җлҹ¬ лң° л•Ң л©ҳл¶• мҳЁлӢӨ.

мҷң мқҙлҹ° кІҢ н•„мҡ”н•ҙ?¶

мғҒнҷ©мқ„ н•ҳлӮҳ мғқк°Ғн•ҙлҙҗ.

  1. л„Ҳк°Җ bank.comм—җ лЎңк·ёмқён–Ҳм–ҙ. м„ём…ҳ мҝ нӮӨк°Җ лёҢлқјмҡ°м Җм—җ мһҲм§Җ.
  2. к·ё мғҒнғңлЎң evil.comм—җ м ‘мҶҚн–Ҳм–ҙ.
  3. evil.comмқҳ JavaScriptк°Җ мқҙлҹ° мҪ”л“ңлҘј мӢӨн–үн•ҙ:
// evil.comмқҳ м•…м„ұ мҠӨнҒ¬лҰҪнҠё
fetch('https://bank.com/api/transfer', {
    method: 'POST',
    body: JSON.stringify({ to: 'кіөкІ©мһҗкі„мўҢ', amount: 1000000 }),
    credentials: 'include'  // мҝ нӮӨ нҸ¬н•Ё!
});

лёҢлқјмҡ°м Җм—җ bank.comмқҳ м„ём…ҳ мҝ нӮӨк°Җ мһҲмңјлӢҲк№Ң, мқҙ мҡ”мІӯм—җ мҝ нӮӨк°Җ мһҗлҸҷмңјлЎң л¶ҷм–ҙ. м•„л¬ҙ м ңн•ң м—Ҷмңјл©ҙ? мҶЎкёҲмқҙ мӢӨн–үлҗңлӢӨ. л„ҲлҸ„ лӘЁлҘҙкІҢ.

мқҙкұё л§үлҠ” кІҢ Same-Origin Policyм•ј.

evil.comм—җм„ң bank.comмңјлЎң ліҙлӮҙлҠ” мҡ”мІӯмқҖ м¶ңмІҳ(Origin)к°Җ лӢӨлҘҙлӢҲк№Ң лёҢлқјмҡ°м Җк°Җ м°ЁлӢЁн•ңлӢӨ.

н•өмӢ¬мқ„ кё°м–өн•ҙ

Same-Origin PolicyлҠ” м•…м„ұ мӮ¬мқҙнҠёк°Җ лӢӨлҘё мӮ¬мқҙнҠёмқҳ APIлҘј н•Ёл¶ҖлЎң нҳём¶ңн•ҳлҠ” кұё л§үлҠ” ліҙм•Ҳ м •мұ…мқҙлӢӨ. мқҙкІҢ м—Ҷмңјл©ҙ мӣ№мқҖ ліҙм•Ҳ м§ҖмҳҘмқҙ лҸј.

Origin(м¶ңмІҳ)мқҙлһҖ¶

Originмқҳ кө¬м„ұ¶

OriginмқҖ scheme + host + port мЎ°н•©мқҙлӢӨ.

https://api.example.com:443
  в”Ӯ         в”Ӯ            в”Ӯ
scheme    host          port

мқҙ 3к°Җм§Җк°Җ м „л¶Җ мқјм№ҳн•ҙм•ј Same-Origin(к°ҷмқҖ м¶ңмІҳ)мқҙм•ј.

Same-Origin нҢҗлі„ мҳҲмӢң¶

кё°мӨҖ URL: https://www.example.com:443/page

л№„көҗ URL Same-Origin? мқҙмң 
https://www.example.com:443/other O scheme, host, port м „л¶Җ мқјм№ҳ (кІҪлЎңлҠ” л¬ҙкҙҖ)

https://api.example.com:443/page X hostк°Җ лӢӨлҰ„ (www vs api)
https://www.example.com:8080/page X portк°Җ лӢӨлҰ„ (443 vs 8080)
https://www.example.com:443/page?q=1 O мҝјлҰ¬мҠӨнҠёл§ҒмқҖ л¬ҙкҙҖ

м„ңлёҢлҸ„л©”мқёлҸ„ лӢӨлҘё Originмқҙм•ј

www.example.comкіј api.example.comмқҖ лӢӨлҘё OriginмқҙлӢӨ. к°ҷмқҖ example.com лҸ„л©”мқёмқҙлқјлҸ„ м„ңлёҢлҸ„л©”мқёмқҙ лӢӨлҘҙл©ҙ Cross-Originмқҙм•ј. мқҙкұ° лӘЁлҘҙл©ҙ "к°ҷмқҖ лҸ„л©”мқёмқёлҚ° мҷң CORS м—җлҹ¬лӮҳм§Җ?" н•ҳкі  мӮҪм§Ҳн•ҙ.

Same-Origin Policy (лҸҷмқј м¶ңмІҳ м •мұ…)¶

м •мқҳ¶

Same-Origin PolicyлҠ” лёҢлқјмҡ°м Җк°Җ к°•м ңн•ҳлҠ” ліҙм•Ҳ м •мұ…мқҙлӢӨ.

лӢӨлҘё Originмқҳ лҰ¬мҶҢмҠӨм—җ лҢҖн•ң JavaScriptмқҳ м ‘к·јмқ„ м ңн•ңн•ңлӢӨ.

м„ңлІ„к°Җ л§үлҠ” кІҢ м•„лӢҲм•ј. лёҢлқјмҡ°м Җк°Җ л§үлҠ” кұ°м•ј.

мқҙкІҢ к°ҖмһҘ л§Һмқҙ н—·к°ҲлҰ¬лҠ” л¶Җл¶„мқҙм•ј.

  1. лёҢлқјмҡ°м Җк°Җ evil.comм—җм„ң bank.comмңјлЎң мҡ”мІӯмқ„ ліҙлӮёлӢӨ
  2. м„ңлІ„лҠ” м •мғҒм ҒмңјлЎң мқ‘лӢөмқ„ лҸҢл ӨмӨҖлӢӨ (м„ңлІ„лҠ” Origin мӢ кІҪ м•Ҳ мҚЁлҸ„ лҗЁ)
  3. лёҢлқјмҡ°м Җк°Җ мқ‘лӢөмқ„ л°ӣкі , Originмқ„ нҷ•мқён•ңлӢӨ
  4. "мқҙ мқ‘лӢөм—җ CORS н—Ҳмҡ© н—ӨлҚ”к°Җ м—Ҷл„Ө? м°ЁлӢЁ!"
  5. лёҢлқјмҡ°м Җк°Җ JavaScriptм—җкІҢ мқ‘лӢөмқ„ м•Ҳ л„ҳкІЁмӨҖлӢӨ

м„ңлІ„лҠ” мқ‘лӢөмқ„ лӢӨ ліҙлғҲм–ҙ. лёҢлқјмҡ°м Җк°Җ к·ё мқ‘лӢөмқ„ JSм—җкІҢ м•Ҳ мЈјлҠ” кұ°м•ј.

м ңн•ңлҗҳлҠ” кІғкіј м ңн•ң м•Ҳ лҗҳлҠ” кІғ¶

Same-Origin Policyк°Җ л§үлҠ” кұҙ JavaScriptмқҳ лҚ°мқҙн„° м ‘к·јмқҙм•ј. лҰ¬мҶҢмҠӨ лЎңл“ң мһҗмІҙк°Җ м „л¶Җ л§үнһҲлҠ” кұҙ м•„лӢҲм•ј.

лҸҷмһ‘ м ңн•ң м—¬л¶Җ м„ӨлӘ…
<img src="лӢӨлҘёOrigin/image.png"> н—Ҳмҡ© мқҙлҜём§Җ лЎңл“ңлҠ” OK
<script src="лӢӨлҘёOrigin/app.js"> н—Ҳмҡ© мҠӨнҒ¬лҰҪнҠё лЎңл“ңлҠ” OK
<link href="лӢӨлҘёOrigin/style.css"> н—Ҳмҡ© CSS лЎңл“ңлҠ” OK
<form action="лӢӨлҘёOrigin/submit"> н—Ҳмҡ© нҸј м „мҶЎмқҖ OK
fetch('лӢӨлҘёOrigin/api/data') м ңн•ң JSлЎң лҚ°мқҙн„° мқҪкё°лҠ” м°ЁлӢЁ
XMLHttpRequest to лӢӨлҘё Origin м ңн•ң AJAX мҡ”мІӯмқҳ мқ‘лӢө мқҪкё° м°ЁлӢЁ

мҷң мқҙлҜём§Җ, мҠӨнҒ¬лҰҪнҠёлҠ” н—Ҳмҡ©н•ҙ?

мӣ№мқҙ мӣҗлһҳ к·ёл ҮкІҢ м„Өкі„лҗҗм–ҙ. CDNм—җм„ң jQuery лЎңл“ңн•ҳкі , лӢӨлҘё лҸ„л©”мқёмқҳ мқҙлҜём§ҖлҘј ліҙм—¬мЈјлҠ” кұҙ мӣ№мқҳ кё°ліё кё°лҠҘмқҙлӢҲк№Ң. л¬ём ңлҠ” JavaScriptк°Җ н”„лЎңк·ёлһҳл°Қм ҒмңјлЎң лҚ°мқҙн„°лҘј мқҪлҠ” кІғмқҙм•ј. к·ёкІҢ н—Ҳмҡ©лҗҳл©ҙ м•…м„ұ JSк°Җ лӢӨлҘё мӮ¬мқҙнҠёмқҳ APIлҘј л§ҲмқҢлҢҖлЎң нҳём¶ңн•  мҲҳ мһҲмңјлӢҲк№Ң.

CORS (Cross-Origin Resource Sharing)¶

м •мқҳ¶

CORSлҠ” м„ңлІ„к°Җ "мқҙ Originм—җм„ң мҳӨлҠ” мҡ”мІӯмқҖ н—Ҳмҡ©н• кІҢ"лқјкі  лӘ…мӢңм ҒмңјлЎң м•Ңл ӨмЈјлҠ” л©”м»ӨлӢҲмҰҳмқҙлӢӨ.

Same-Origin PolicyлҠ” кё°ліём ҒмңјлЎң лӢӨ л§үм•„. н•ҳм§Җл§Ң мӢӨл¬ҙм—җм„ңлҠ” лӢӨлҘё Originм—җм„ң APIлҘј нҳём¶ңн•ҙм•ј н•ҳлҠ” кІҪмҡ°к°Җ л§Һмһ–м•„.

  • н”„лЎ нҠём—”л“ң(localhost:3000)к°Җ л°ұм—”л“ң(localhost:8080) API нҳём¶ң
  • www.example.comм—җм„ң api.example.com API нҳём¶ң
  • лӘЁл°”мқј м•ұмқҙ API м„ңлІ„ нҳём¶ң

мқҙлҹ° м •мғҒм Ғмқё мҡ”мІӯмқ„ н—Ҳмҡ©н•ҳл Өл©ҙ м„ңлІ„к°Җ CORS н—ӨлҚ”лҘј ліҙлӮҙм•ј н•ңлӢӨ.

CORS кҙҖл Ё н—ӨлҚ”¶

м„ңлІ„к°Җ мқ‘лӢөм—җ нҸ¬н•Ён•ҳлҠ” н—ӨлҚ”л“Ө:

Access-Control-Allow-Origin¶

Access-Control-Allow-Origin: https://www.example.com

м–ҙл–Ө Originмқ„ н—Ҳмҡ©н• м§Җ м§Җм •н•ңлӢӨ.

к°’ мқҳлҜё
https://www.example.com мқҙ Originл§Ң н—Ҳмҡ©
* лӘЁл“  Origin н—Ҳмҡ© (мЈјмқҳ!)

мҷҖмқјл“ңм№ҙл“ң(*)лҠ” мң„н—ҳн•  мҲҳ мһҲм–ҙ

Access-Control-Allow-Origin: *лҠ” лӘЁл“  мӮ¬мқҙнҠём—җм„ң м ‘к·ј к°ҖлҠҘн•ҳлӢӨлҠ” лң»мқҙм•ј. кіөк°ң APIл©ҙ кҙңм°®м§Җл§Ң, мқёмҰқмқҙ н•„мҡ”н•ң APIм—җ *лҘј м“°л©ҙ ліҙм•Ҳ кө¬л©Қ. нҠ№нһҲ credentials: 'include'(мҝ нӮӨ нҸ¬н•Ё)мҷҖ *лҠ” лҸҷмӢңм—җ м“ё мҲҳ м—Ҷм–ҙ. лёҢлқјмҡ°м Җк°Җ кұ°л¶Җн•ңлӢӨ.

Access-Control-Allow-Methods¶

Access-Control-Allow-Methods: GET, POST, PUT, DELETE

м–ҙл–Ө HTTP л©”м„ңл“ңлҘј н—Ҳмҡ©н• м§Җ м§Җм •н•ңлӢӨ.

Access-Control-Allow-Headers¶

Access-Control-Allow-Headers: Content-Type, Authorization

м–ҙл–Ө мҡ”мІӯ н—ӨлҚ”лҘј н—Ҳмҡ©н• м§Җ м§Җм •н•ңлӢӨ. кё°ліё н—ӨлҚ”(Accept, Content-Language л“ұ) мҷём—җ м»ӨмҠӨн…Җ н—ӨлҚ”лҘј м“ё л•Ң н•„мҡ”.

Access-Control-Allow-Credentials¶

Access-Control-Allow-Credentials: true

мҝ нӮӨлҘј нҸ¬н•Ён•ң мҡ”мІӯмқ„ н—Ҳмҡ©н• м§Җ м§Җм •н•ңлӢӨ.

мқҙкІҢ trueл©ҙ Access-Control-Allow-Originм—җ *лҘј лӘ» мҚЁ. м •нҷ•н•ң Originмқ„ лӘ…мӢңн•ҙм•ј н•ҙ.

Access-Control-Max-Age¶

Access-Control-Max-Age: 3600

Preflight мқ‘лӢөмқ„ мәҗмӢұн•  мӢңк°„(мҙҲ). мқҙ мӢңк°„ лҸҷм•ҲмқҖ к°ҷмқҖ мҡ”мІӯм—җ лҢҖн•ҙ PreflightлҘј лӢӨмӢң ліҙлӮҙм§Җ м•ҠлҠ”лӢӨ.

м„ңлІ„к°Җ Access-Control-Allow-Originм—җ мҡ”мІӯн•ң Originмқ„ нҸ¬н•ЁмӢңнӮӨл©ҙ, лёҢлқјмҡ°м Җк°Җ мқ‘лӢөмқ„ JavaScriptм—җ м „лӢ¬н•ҙмӨҖлӢӨ.

Preflight мҡ”мІӯ¶

мқҙкІҢ лӯҗм•ј¶

нҠ№м • мЎ°кұҙмқҳ мҡ”мІӯмқҖ лёҢлқјмҡ°м Җк°Җ ліё мҡ”мІӯмқ„ ліҙлӮҙкё° м „м—җ OPTIONS мҡ”мІӯмқ„ лЁјм Җ ліҙлӮёлӢӨ. мқҙкұё Preflight мҡ”мІӯмқҙлқјкі  н•ҙ.

6мһҘм—җм„ң OPTIONS л©”м„ңл“ң л°°мӣ м§Җ? "мқҙ м„ңлІ„к°Җ м–ҙл–Ө л©”м„ңл“ңлҘј м§Җмӣҗн•ҳлҠ”м§Җ нҷ•мқён•ҳлҠ” л©”м„ңл“ң"лқјкі  н–Ҳм–ҙ. CORSм—җм„ң л°”лЎң м—¬кё°м„ң м“°мқёлӢӨ.

"лӮҙк°Җ мқҙлҹ° мҡ”мІӯмқ„ ліҙлӮҙлҸ„ лҗҳкІ лӢҲ?" н•ҳкі  лЁјм Җ л¬јм–ҙліҙлҠ” кұ°м•ј.

Simple Request vs Preflight Request¶

лӘЁл“  мҡ”мІӯм—җ Preflightк°Җ л¶ҷлҠ” кұҙ м•„лӢҲм•ј. Simple Request мЎ°кұҙмқ„ л§ҢмЎұн•ҳл©ҙ Preflight м—Ҷмқҙ л°”лЎң ліё мҡ”мІӯмқ„ ліҙлӮёлӢӨ.

Simple Request мЎ°кұҙ (м „л¶Җ л§ҢмЎұн•ҙм•ј н•Ё):

мЎ°кұҙ н—Ҳмҡ© к°’
л©”м„ңл“ң GET, HEAD, POSTл§Ң
Content-Type application/x-www-form-urlencoded, multipart/form-data, text/plainл§Ң
н—ӨлҚ” Accept, Accept-Language, Content-Language, Content-Type л“ұ кё°ліё н—ӨлҚ”л§Ң

мқҙ мЎ°кұҙмқ„ н•ҳлӮҳлқјлҸ„ лІ—м–ҙлӮҳл©ҙ Preflightк°Җ л°ңмғқн•ңлӢӨ.

мӢӨл¬ҙм—җм„ң Preflightк°Җ л°ңмғқн•ҳлҠ” нқ”н•ң кІҪмҡ°

@Configuration
@EnableWebSecurity
public class SecurityConfig {

    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .cors(cors -> cors.configurationSource(corsConfigurationSource()))
            // ... лӢӨлҘё м„Өм •
        ;
        return http.build();
    }

    @Bean
    public CorsConfigurationSource corsConfigurationSource() {
        CorsConfiguration config = new CorsConfiguration();
        config.setAllowedOrigins(List.of("http://localhost:3000"));
        config.setAllowedMethods(List.of("GET", "POST", "PUT", "DELETE"));
        config.setAllowedHeaders(List.of("*"));
        config.setAllowCredentials(true);

        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/api/**", config);
        return source;
    }
}

мЈјмқҳмӮ¬н•ӯ / н•Ём •¶

нқ”н•ң мӢӨмҲҳл“Ө

1. "CORSлҠ” н”„лЎ нҠём—”л“ң л¬ём ңлӢӨ"

м•„лӢҲм•ј. CORS н—ӨлҚ”лҠ” м„ңлІ„к°Җ ліҙлӮҙлҠ” кІғмқҙм•ј. н”„лЎ нҠём—”л“ңм—җм„ң н•  мҲҳ мһҲлҠ” кұҙ мҡ”мІӯмқ„ ліҙлӮҙлҠ” кІғлҝҗ. н—Ҳмҡ© м—¬л¶ҖлҠ” м„ңлІ„к°Җ кІ°м •н•ҳкі , м°ЁлӢЁ м—¬л¶ҖлҠ” лёҢлқјмҡ°м Җк°Җ кІ°м •н•ҙ.

2. "Postmanм—җм„ңлҠ” лҗҳлҠ”лҚ° лёҢлқјмҡ°м Җм—җм„ңлҠ” м•Ҳ лҸјмҡ”"

л§һм•„. PostmanмқҖ лёҢлқјмҡ°м Җк°Җ м•„лӢҲлӢҲк№Ң Same-Origin Policyк°Җ м—Ҷм–ҙ. CORSлҠ” лёҢлқјмҡ°м Җ м „мҡ© ліҙм•Ҳ м •мұ…мқҙм•ј. Postman, curl, м„ңлІ„ к°„ нҶөмӢ м—җм„ңлҠ” CORSк°Җ м Ғмҡ© м•Ҳ лҸј. мқҙкІҢ л°”лЎң "м„ңлІ„к°Җ л§үлҠ” кІҢ м•„лӢҲлқј лёҢлқјмҡ°м Җк°Җ л§үлҠ” кІғ"мқҳ мҰқкұ°м•ј.

3. "Access-Control-Allow-Origin: * н•ҳл©ҙ лӢӨ лҗҳмһ–м•„"
