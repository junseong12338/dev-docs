9. SSLкіј н”„лЎқмӢң - BetaВ¶

1. н•өмӢ¬ к°ңл…җ - "мқҙкІҢ лӯҗм•ј?"В¶

мқҙкұ° мҷң м•Ңм•„м•ј н•ҳлғҗл©ҙ, ngrokмқҙ мҷң мҡ°лҰ¬ л¬ём ңлҘј н•ҙкІ°н•ҙмЈјлҠ”м§Җ м •нҷ•нһҲ мқҙн•ҙн•ҳл Өл©ҙ "SSL мў…лЈҢ"лқјлҠ” к°ңл…җмқ„ м•Ңм•„м•ј н•ҙ. ZoomмқҖ HTTPSлЎң Webhookмқ„ ліҙлӮҙлҠ”лҚ°, мҡ°лҰ¬ Spring BootлҠ” HTTPл§Ң л°ӣм•„. мқҙ мӮ¬мқҙм—җм„ң лҲ„к°Җ HTTPSлҘј лІ—кІЁмЈјлҠ” кұ°м•ј? к·ёкІҢ SSL мў…лЈҢм•ј.

л№„мң : кіөн•ӯ ліҙм•ҲкІҖмғүВ¶

н•ҙмҷё нғқл°°(HTTPS мҡ”мІӯ)к°Җ н•ңкөӯм—җ л“Өм–ҙмҳ¬ л•ҢлҘј мғқк°Ғн•ҙлҙҗ.

  • нғқл°°к°Җ кіөн•ӯ(н”„лЎқмӢң)м—җ лҸ„м°©н•ҳл©ҙ м„ёкҙҖ(SSL мў…лЈҢ)м—җм„ң нҸ¬мһҘмқ„ лңҜм–ҙ
  • лӮҙмҡ©л¬јмқ„ нҷ•мқён•ҳкі 
  • көӯлӮҙ л°°мҶЎ(HTTP)мқҖ мқјл°ҳ нғқл°°лЎң ліҙлӮҙ

л°–м—җм„ң л“Өм–ҙмҳ¬ л•ҢлҠ” ліҙм•Ҳ нҸ¬мһҘ(HTTPS)мқҙ н•„мҲҳм§Җл§Ң, көӯлӮҙ л°°мҶЎ(лӮҙл¶Җ л„ӨнҠёмӣҢнҒ¬)мқҖ мқјл°ҳ нҸ¬мһҘ(HTTP)мңјлЎң м¶©л¶„н•ң кұ°м•ј.

л№„мң лҠ” м—¬кё°к№Ңм§Җ. ліём§Ҳ к°Җмһҗ.

SSL мў…лЈҢ(SSL Termination) м •мқҳ

лҰ¬лІ„мҠӨ н”„лЎқмӢңк°Җ нҒҙлқјмқҙм–ёнҠёмҷҖмқҳ HTTPS м—°кІ°мқ„ мІҳлҰ¬(мў…лЈҢ)н•ҳкі , л°ұм—”л“ң м„ңлІ„м—җлҠ” HTTPлЎң м „лӢ¬н•ҳлҠ” кө¬мЎ°. н”„лЎқмӢңк°Җ SSL/TLS м•”нҳёнҷ”лҘј лІ—кё°лҠ” м§Җм җмқҙ "SSL мў…лЈҢ м§Җм җ"мқҙлӢӨ.

2. лҸҷмһ‘ мӣҗлҰ¬ - "м–ҙл–»кІҢ лҸҢм•„к°Җ?"В¶

SSL мІҳлҰ¬ л°©мӢқ 3к°Җм§ҖВ¶

лҰ¬лІ„мҠӨ н”„лЎқмӢңм—җм„ң SSLмқ„ мІҳлҰ¬н•ҳлҠ” л°©мӢқмқҖ нҒ¬кІҢ 3к°Җм§Җм•ј.

1. SSL мў…лЈҢ (SSL Termination)В¶

к°ҖмһҘ нқ”н•ң л°©мӢқ. н”„лЎқмӢңк°Җ HTTPSлҘј лІ—кё°кі  л°ұм—”л“ңм—җлҠ” HTTPлЎң.

sequenceDiagram
    participant C as нҒҙлқјмқҙм–ёнҠё
    participant P as н”„лЎқмӢң (мқёмҰқм„ң ліҙмң )
    participant B as л°ұм—”л“ң м„ңлІ„

    C->>P: HTTPS мҡ”мІӯ (м•”нҳёнҷ”)
    Note over P: SSL мў…лЈҢ - ліөнҳёнҷ”
    P->>B: HTTP мҡ”мІӯ (нҸүл¬ё)
    B-->>P: HTTP мқ‘лӢө (нҸүл¬ё)
    Note over P: м•”нҳёнҷ”
    P-->>C: HTTPS мқ‘лӢө (м•”нҳёнҷ”)

2. SSL нҢЁмҠӨмҠӨлЈЁ (SSL Passthrough)В¶

н”„лЎқмӢңк°Җ SSLмқ„ кұҙл“ңлҰ¬м§Җ м•Ҡкі , м•”нҳёнҷ”лҗң мұ„лЎң л°ұм—”л“ңм—җ к·ёлҢҖлЎң м „лӢ¬.

sequenceDiagram
    participant C as нҒҙлқјмқҙм–ёнҠё
    participant P as н”„лЎқмӢң (мқёмҰқм„ң м—ҶмқҢ)
    participant B as л°ұм—”л“ң м„ңлІ„ (мқёмҰқм„ң ліҙмң )

    C->>P: HTTPS мҡ”мІӯ (м•”нҳёнҷ”)
    Note over P: к·ёлҢҖлЎң м „лӢ¬ (м•Ҳ кұҙл“ңлҰј)
    P->>B: HTTPS мҡ”мІӯ (м•”нҳёнҷ” мң м§Җ)
    B-->>P: HTTPS мқ‘лӢө (м•”нҳёнҷ”)
    P-->>C: HTTPS мқ‘лӢө (м•”нҳёнҷ”)

3. SSL лёҢлҰҝм§• (SSL Bridging)В¶

н”„лЎқмӢңк°Җ SSLмқ„ лІ—кё°кі , л°ұм—”л“ңм—җ лӢӨмӢң мғҲлЎңмҡҙ SSLлЎң м—°кІ°.

sequenceDiagram
    participant C as нҒҙлқјмқҙм–ёнҠё
    participant P as н”„лЎқмӢң (мҷёл¶Җ мқёмҰқм„ң)
    participant B as л°ұм—”л“ң м„ңлІ„ (лӮҙл¶Җ мқёмҰқм„ң)

    C->>P: HTTPS мҡ”мІӯ (мҷёл¶Җ мқёмҰқм„ңлЎң м•”нҳёнҷ”)
    Note over P: ліөнҳёнҷ” нӣ„ лӢӨмӢң м•”нҳёнҷ”
    P->>B: HTTPS мҡ”мІӯ (лӮҙл¶Җ мқёмҰқм„ңлЎң мһ¬м•”нҳёнҷ”)
    B-->>P: HTTPS мқ‘лӢө
    P-->>C: HTTPS мқ‘лӢө

3к°Җм§Җ л°©мӢқ л№„көҗВ¶

н•ӯлӘ© SSL мў…лЈҢ SSL нҢЁмҠӨмҠӨлЈЁ SSL лёҢлҰҝм§•
н”„лЎқмӢңм—җ мқёмҰқм„ң н•„мҡ” л¶Ҳн•„мҡ” н•„мҡ” (мҷёл¶Җмҡ©)
л°ұм—”л“ңм—җ мқёмҰқм„ң л¶Ҳн•„мҡ” н•„мҡ” н•„мҡ” (лӮҙл¶Җмҡ©)
н”„лЎқмӢңк°Җ лӮҙмҡ© ліј мҲҳ мһҲлӮҳ к°ҖлҠҘ л¶Ҳк°ҖлҠҘ к°ҖлҠҘ
лӮҙл¶Җ кө¬к°„ м•”нҳёнҷ” м•Ҳ лҗЁ (HTTP) лҗЁ лҗЁ
м„ұлҠҘ л°ұм—”л“ң л¶Җн•ҳ к°җмҶҢ ліҖнҷ” м—ҶмқҢ н”„лЎқмӢң + л°ұм—”л“ң л‘ҳ лӢӨ л¶Җн•ҳ
мӮ¬мҡ© л№ҲлҸ„ к°ҖмһҘ нқ”н•Ё ліҙм•Ҳ м—„кІ©н•  л•Ң л“ңл¬ҫ
лЎңл“ңл°ёлҹ°мӢұ к°ҖлҠҘ (лӮҙмҡ© кё°л°ҳ) м ңн•ңм Ғ (L4л§Ң) к°ҖлҠҘ

мӢӨм „м—җм„ң лӯҳ м“°лӮҳ

лҢҖл¶Җл¶„ SSL мў…лЈҢлҘј м“ҙлӢӨ. мқҙмң : мқёмҰқм„ң кҙҖлҰ¬к°Җ н”„лЎқмӢң н•ң кіім—җм„ң лҒқлӮҳкі , л°ұм—”л“ң м„ңлІ„мқҳ SSL мІҳлҰ¬ л¶Җн•ҳк°Җ м—Ҷкі , н”„лЎқмӢңм—җм„ң мҡ”мІӯ лӮҙмҡ©мқ„ ліҙкі  лқјмҡ°нҢ…н•  мҲҳ мһҲм–ҙм„ң.

3. мҪ”л“ң/м„Өм •мңјлЎң ліҙмһҗВ¶

nginx SSL мў…лЈҢ м„Өм •В¶

server {
    listen 443 ssl;                                    # 443 нҸ¬нҠём—җм„ң SSLлЎң
    server_name nexclass.knu10.cc;

    # мқёмҰқм„ң нҢҢмқј (н”„лЎқмӢңм—җ м„Өм№ҳ)
    ssl_certificate /etc/letsencrypt/live/nexclass.knu10.cc/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/nexclass.knu10.cc/privkey.pem;

    # SSL м„Өм •
    ssl_protocols TLSv1.2 TLSv1.3;                    # TLS 1.2 мқҙмғҒл§Ң
    ssl_ciphers HIGH:!aNULL:!MD5;                      # м•Ҳм „н•ң м•”нҳёнҷ”л§Ң

    location / {
        proxy_pass http://localhost:8082;               # л°ұм—”л“ңм—җлҠ” HTTPлЎң!
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-Proto $scheme;     # HTTPSмқём§Җ м•Ңл ӨмӨҢ
    }
}

# HTTP вҶ’ HTTPS лҰ¬лӢӨмқҙл үнҠё
server {
    listen 80;
    server_name nexclass.knu10.cc;
    return 301 https://$host$request_uri;               # к°•м ңлЎң HTTPSлЎң
}

  3. proxy_pass http://... -- л°ұм—”л“ңм—җлҠ” HTTPлЎң м „лӢ¬ (SSL мў…лЈҢ!)
  4. X-Forwarded-Proto -- л°ұм—”л“ңм—җкІҢ "мӣҗлһҳ HTTPSлЎң мҷ”м–ҙ"лқјкі  м•Ңл ӨмӨҢ

4. мЈјмқҳмӮ¬н•ӯ/н•Ём •В¶

лӮҙл¶Җ кө¬к°„ HTTPмқҳ ліҙм•Ҳ л¬ём ңВ¶

SSL мў…лЈҢмқҳ л§№м җ

н”„лЎқмӢң вҶ” л°ұм—”л“ң кө¬к°„мқҙ HTTP(нҸүл¬ё)м•ј. к°ҷмқҖ м„ңлІ„(localhost)л©ҙ кҙңм°®м§Җл§Ң, н”„лЎқмӢңмҷҖ л°ұм—”л“ңк°Җ лӢӨлҘё м„ңлІ„мқҙкі  к·ё мӮ¬мқҙ л„ӨнҠёмӣҢнҒ¬лҘј мӢ лў°н•  мҲҳ м—ҶлӢӨл©ҙ мң„н—ҳн•ҙ.

кө¬к°„ н”„лЎңнҶ мҪң ліҙм•Ҳ
нҒҙлқјмқҙм–ёнҠё вҶ” н”„лЎқмӢң HTTPS (м•”нҳёнҷ”) м•Ҳм „
н”„лЎқмӢң вҶ” л°ұм—”л“ң (к°ҷмқҖ м„ңлІ„) HTTP (нҸүл¬ё) м•Ҳм „ (localhostлӢҲк№Ң)
н”„лЎқмӢң вҶ” л°ұм—”л“ң (лӢӨлҘё м„ңлІ„) HTTP (нҸүл¬ё) мң„н—ҳн•  мҲҳ мһҲмқҢ

мҡ°лҰ¬ NexClassлҠ” nginxмҷҖ Spring Bootк°Җ к°ҷмқҖ м„ңлІ„(localhost)м—җм„ң лҸҢм•„к°ҖлӢҲк№Ң SSL мў…лЈҢлЎң м¶©л¶„н•ҙ. к·јлҚ° л§Ңм•Ҫ л°ұм—”л“ңк°Җ лӢӨлҘё м„ңлІ„м—җ мһҲлӢӨл©ҙ SSL лёҢлҰҝм§•мқ„ кі л Өн•ҙм•ј н•ҙ.

X-Forwarded-Proto н—ӨлҚ”В¶

SSL мў…лЈҢлҘј н•ҳл©ҙ л°ұм—”л“ң м„ңлІ„лҠ” "мқҙ мҡ”мІӯмқҙ мӣҗлһҳ HTTPлЎң мҳЁ кұҙм§Җ HTTPSлЎң мҳЁ кұҙм§Җ" лӘ°лқј. н”„лЎқмӢңн•ңн…Ң HTTPлЎң л°ӣмңјлӢҲк№Ң н•ӯмғҒ HTTPлЎң мқёмӢқн•ҙ.

мқҙкІҢ мҷң л¬ём ңлғҗ:

// Spring Bootм—җм„ң лҰ¬лӢӨмқҙл үнҠё URL мғқм„ұн•  л•Ң
// SSL мў…лЈҢ нҷҳкІҪм—җм„ң мқҙл ҮкІҢ лҗҳл©ҙ...
String redirectUrl = request.getRequestURL().toString();
// кІ°кіј: http://nexclass.knu10.cc/dashboard  <-- HTTPлЎң лӮҳмҳҙ!
// мӣҗлһҳлҠ” httpsлЎң лӮҳмҷҖм•ј н•ҳлҠ”лҚ°!

к·ёлһҳм„ң X-Forwarded-Proto н—ӨлҚ”лЎң мӣҗлһҳ н”„лЎңнҶ мҪңмқ„ м•Ңл ӨмЈјкі , Spring Bootм—җм„ң мқҙкұё мқёмӢқн•ҳлҸ„лЎқ м„Өм •н•ҙм•ј н•ҙ.

proxy_set_header X-Forwarded-Proto $scheme;  # https лҳҗлҠ” http

# application.properties
server.forward-headers-strategy=native

ngrok мқҳмЎҙмқҳ л¬ём ңм җВ¶

л¬ём ң м„ӨлӘ…
URL ліҖкІҪ ngrok мһ¬мӢңмһ‘л§ҲлӢӨ URLмқҙ л°”лҖң вҶ’ Zoom Marketplaceм—җ мһ¬л“ұлЎқ н•„мҡ”
н„°лҜёл„җ мң м§Җ ngrok н”„лЎңм„ёмҠӨ мЈҪмңјл©ҙ Webhook мҲҳмӢ  л¶Ҳк°Җ
л¬ҙлЈҢ м ңн•ң мҡ”мІӯ мҲҳ м ңн•ң, м»ӨмҠӨн…Җ лҸ„л©”мқё л¶Ҳк°Җ
