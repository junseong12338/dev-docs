01. IP мЈјмҶҢмҷҖ нҸ¬нҠё - мқён„°л„·мқҳ мўҢн‘ң мӢңмҠӨн…ң¶

лӮңмқҙлҸ„: Alpha

мқён”„лқјмқҳ к°ҖмһҘ кё°ліё. мқҙкұ° лӘЁлҘҙл©ҙ л’Өм—җ лӮҳмҳӨлҠ” кұ° н•ҳлӮҳлҸ„ мқҙн•ҙ лӘ» н•ҙ.

IP мЈјмҶҢк°Җ лӯҗлғҗ¶

"м§‘ мЈјмҶҢ"лқјкі  л№„мң н•ҳлҠ” көҗмһ¬к°Җ л§ҺмқҖлҚ°, л№„мң лҠ” кұ°кё°к№Ңм§Җл§Ң м“°мһҗ. ліём§ҲмқҖ мқҙкұ°м•ј.

IP мЈјмҶҢмқҳ ліём§Ҳ

л„ӨнҠёмӣҢнҒ¬м—җ м—°кІ°лҗң мһҘм№ҳлҘј мӢқлі„н•ҳлҠ” кі мң н•ң мҲ«мһҗ мЈјмҶҢ. мқён„°л„·м—җм„ң "л„Ҳн•ңн…Ң лҚ°мқҙн„° ліҙлӮјкІҢ" н•  л•Ң, м–ҙл””лЎң ліҙлӮјм§Җ м•Ңм•„м•ј н•ҳмһ–м•„. к·ёкІҢ IPм•ј.

IPv4 мЈјмҶҢ нҳ•мӢқ¶

49.247.45.181
  • 4к°ңмқҳ мҲ«мһҗлҘј .мңјлЎң кө¬л¶„
  • к°Ғ мҲ«мһҗлҠ” 0~255 лІ”мң„ (8л№„нҠё = 1л°”мқҙнҠё, мҙқ 32л№„нҠё)
  • м „ м„ёкі„м—җм„ң мң мқјн•ң мЈјмҶҢ (кіөмқё IPмқҳ кІҪмҡ°)

кіөмқё IP vs лӮҙл¶Җ(мӮ¬м„Ө) IP¶

мқҙкұ° кө¬л¶„ лӘ» н•ҳл©ҙ мӮҪм§Ҳмқҳ мӢңмһ‘мқҙм•ј

"м„ңлІ„ IPк°Җ лӯҗмҳҲмҡ”?" л¬јм—Ҳмқ„ л•Ң м–ҙл–Ө IPлҘј л§җн•ҳлҠ”м§Җ лӘЁлҘҙл©ҙ лҒқмһҘмқҙм•ј.

нҸ¬нҠё(Port)к°Җ лӯҗлғҗ¶

IP мЈјмҶҢк°Җ кұҙл¬ј мЈјмҶҢлқјл©ҙ, нҸ¬нҠёлҠ” к·ё кұҙл¬ј м•Ҳмқҳ нҳёмҲҳм•ј. м—¬кё°к№Ңм§Җк°Җ л№„мң кі , ліём§ҲмқҖ мқҙкұ°м•ј.

нҸ¬нҠёмқҳ ліём§Ҳ

н•ҳлӮҳмқҳ IP мЈјмҶҢ(м„ңлІ„)м—җм„ң м—¬лҹ¬ н”„лЎңк·ёлһЁмқҙ лҸҷмӢңм—җ лҸҢм•„к°Ҳ л•Ң, м–ҙл–Ө н”„лЎңк·ёлһЁн•ңн…Ң лҚ°мқҙн„°лҘј м „лӢ¬н• м§Җ кө¬л¶„н•ҳлҠ” лІҲнҳё. лІ”мң„: 0 ~ 65535 (16л№„нҠё)

мҷң нҸ¬нҠёк°Җ н•„мҡ”н•ҙ?¶

м„ңлІ„ н•ҳлӮҳм—җ н”„лЎңк·ёлһЁмқҙ н•ҳлӮҳл§Ң лҸҢм•„к°Җл©ҙ нҸ¬нҠёк°Җ н•„мҡ” м—Ҷм–ҙ. к·јлҚ° нҳ„мӢӨмқҖ?

мҡ°лҰ¬ WAS м„ңлІ„ (10.7.1.52) нҳ„нҷ©

м„ңлІ„ 1лҢҖм—җ н”„лЎңк·ёлһЁ 4к°ңк°Җ лҸҷмӢңм—җ лҸҢм•„к°Җкі  мһҲм–ҙ:

нҸ¬нҠё н”„лЎңк·ёлһЁ м„ӨлӘ…
:8080 LMS (www) н•ҷмҠөкҙҖлҰ¬мӢңмҠӨн…ң л©”мқё
:8081 LMS (н•ҷмӮ¬) н•ҷмӮ¬ лҚ°мқҙн„° API
:8082 NexClass нҷ”мғҒк°•мқҳ мһҗмІҙ мҶ”лЈЁм…ҳ
:8090 HUB н—ҲлёҢ мӢңмҠӨн…ң

к°ҷмқҖ IP 10.7.1.52мқёлҚ°, нҸ¬нҠё лІҲнҳёлЎң "л„Ҳн•ңн…Ң ліҙлӮҙлҠ” кұ°м•ј"лҘј кө¬л¶„н•ҳлҠ” кұ°м§Җ.

Well-Known нҸ¬нҠё (мҷёмӣҢ)¶

мқҙкұҙ кұҚ мҷёмӣҢм•ј н•ҙ. м•Ҳ мҷёмҡ°л©ҙ л§ӨлІҲ кІҖмғүн•ҙм•ј лҸј.

нҸ¬нҠё н”„лЎңнҶ мҪң/м„ңл№„мҠӨ м„ӨлӘ…
80 HTTP мӣ№ кё°ліё нҸ¬нҠё (лёҢлқјмҡ°м Җк°Җ мһҗлҸҷмңјлЎң л¶ҷмһ„)
443 HTTPS м•”нҳёнҷ”лҗң мӣ№ кё°ліё нҸ¬нҠё
22 SSH мӣҗкІ© м„ңлІ„ м ‘мҶҚ
3306 MySQL/MariaDB лҚ°мқҙн„°лІ мқҙмҠӨ
5432 PostgreSQL лҚ°мқҙн„°лІ мқҙмҠӨ
8080 мӮ¬мҡ©мһҗ м •мқҳ к°ңл°ңмҡ© мӣ№ м„ңлІ„м—җм„ң л§Һмқҙ м”Җ
8082 мӮ¬мҡ©мһҗ м •мқҳ мҡ°лҰ¬ NexClassк°Җ м—¬кё°

80кіј 443мқҙ мҷң мӨ‘мҡ”н•ҳлғҗ

лёҢлқјмҡ°м Җм—җм„ң https://nexclass.knu10.ccлҘј м№ҳл©ҙ, лёҢлқјмҡ°м Җк°Җ мһҗлҸҷмңјлЎң 443лІҲ нҸ¬нҠёлЎң м—°кІ°н•ҙ. http://nexclass.knu10.ccлҘј м№ҳл©ҙ 80лІҲ нҸ¬нҠёлЎң м—°кІ°н•ҳкі . мҰү, мӮ¬мҡ©мһҗлҠ” нҸ¬нҠё лІҲнҳёлҘј м•Ҳ міҗлҸ„ лҗҳлҠ” кұ°м•ј. лёҢлқјмҡ°м Җк°Җ м•Ңм•„м„ң л¶ҷм—¬мЈјлӢҲк№Ң.

нҸ¬нҠё нҷ•мқё лӘ…л №м–ҙ¶

мӢӨм „: URLмқ„ л¶„н•ҙн•ҙліҙмһҗ¶

https://nexclass.knu10.cc/webhook/zoom -- мқҙ URLм—җм„ң IPмҷҖ нҸ¬нҠёк°Җ м–ҙл””м—җ мҲЁм–ҙмһҲмқ„к№Ң?

URL л¶Җл¶„ мқҳлҜё мҲЁкІЁм§„ к°’
https:// н”„лЎңнҶ мҪң (HTTPS) нҸ¬нҠё 443 (мғқлһөлҗЁ)
nexclass.knu10.cc лҸ„л©”мқё IP 49.247.41.134 (DNSк°Җ ліҖнҷҳ)
/webhook/zoom кІҪлЎң NexClass м•ұ лӮҙл¶Җ лқјмҡ°нҢ…

мһ к№җ, NexClassлҠ” 8082мқёлҚ° мҷң 443мқҙм•ј?

мўӢмқҖ м§Ҳл¬ёмқҙм•ј. мқҙкұҙ 02мһҘ(DNS)кіј 05мһҘ(Nginx лҰ¬лІ„мҠӨ н”„лЎқмӢң)м—җм„ң м„ӨлӘ…н•ҙ. лҜёлҰ¬ н•ң мӨ„л§Ң л§җн•ҳл©ҙ: Nginxк°Җ 443мңјлЎң л°ӣм•„м„ң 8082лЎң м „лӢ¬н•ҙмЈјлҠ” кұ°м•ј.

IP мЈјмҶҢ л„ӨнҠёмӣҢнҒ¬м—җм„ң мһҘм№ҳлҘј м°ҫлҠ” мЈјмҶҢ
кіөмқё IP мқён„°л„·м—җм„ң м ‘к·ј к°ҖлҠҘн•ң мң мқјн•ң мЈјмҶҢ
лӮҙл¶Җ IP к°ҷмқҖ л„ӨнҠёмӣҢнҒ¬ м•Ҳм—җм„ңл§Ң м“°лҠ” мЈјмҶҢ (10.x, 192.168.x л“ұ)
нҸ¬нҠё к°ҷмқҖ м„ңлІ„м—җм„ң м—¬лҹ¬ н”„лЎңк·ёлһЁмқ„ кө¬л¶„н•ҳлҠ” лІҲнҳё (0~65535)

нҷ•мқё л¬ём ң¶

Q1. мҡ°лҰ¬ WAS м„ңлІ„мқҳ кіөмқё IPмҷҖ лӮҙл¶Җ IPлҘј к°Ғк°Ғ л§җн•ҙлҙҗ. к·ёлҰ¬кі  Zoom Webhookмқҙ NexClassм—җ мҡ”мІӯмқ„ ліҙлӮҙл Өл©ҙ м–ҙл–Ө IPлҘј мҚЁм•ј н•ҙ?

Q2. sudo lsof -i :8082 лӘ…л №м–ҙлҘј мӢӨн–үн–ҲлҠ”лҚ° м•„л¬ҙкІғлҸ„ м•Ҳ лӮҳмҷ”м–ҙ. мқҙкІҢ л¬ҙмҠЁ лң»мқҙм•ј?

Q3. мҡ°лҰ¬ WAS м„ңлІ„м—җм„ң LMSлҠ” 8080, NexClassлҠ” 8082, HUBлҠ” 8090 нҸ¬нҠёлҘј м“°кі  мһҲм–ҙ. л§Ңм•Ҫ нҸ¬нҠёлқјлҠ” к°ңл…җмқҙ м—ҶлӢӨл©ҙ м–ҙл–Ө л¬ём ңк°Җ мғқкІЁ?

Q4. мӮ¬мҡ©мһҗк°Җ лёҢлқјмҡ°м Җм—җ https://nexclass.knu10.ccлҘј мһ…л Ҙн–Ҳм–ҙ. мқҙл•Ң лёҢлқјмҡ°м Җк°Җ мӢӨм ңлЎң м—°кІ°н•ҳлҠ” нҸ¬нҠё лІҲнҳёлҠ” лӯҗм•ј? к·ёлҰ¬кі  мҷң URLм—җ нҸ¬нҠёк°Җ м•Ҳ ліҙм—¬?

Q5. hostname -IмҷҖ curl ifconfig.meмқҳ кІ°кіјк°Җ лӢӨлҘё мқҙмң лҘј м„ӨлӘ…н•ҙлҙҗ.

м •лӢө ліҙкё°

A1. кіөмқё IP: 49.247.45.181, лӮҙл¶Җ IP: 10.7.1.52. Zoom WebhookмқҖ мқён„°л„·мқ„ нҶөн•ҙ мҳӨлӢҲк№Ң кіөмқё IPлҘј мҚЁм•ј н•ҙ. к·јлҚ° мӢӨм ңлЎңлҠ” кіөмқё IPлҘј м§Ғм ‘ м“°лҠ” кІҢ м•„лӢҲлқј лҸ„л©”мқё(nexclass.knu10.cc)мқ„ м“°кі , DNSк°Җ WEB м„ңлІ„ IP(49.247.41.134)лЎң ліҖнҷҳн•ҙмӨҳ. WAS кіөмқё IPм—җ м§Ғм ‘ м ‘к·јн•ҳлҠ” кІҢ м•„лӢҲлқј WEB м„ңлІ„(Nginx)лҘј кұ°міҗм„ң лӮҙл¶Җл§қмңјлЎң м „лӢ¬лҗҳлҠ” кө¬мЎ°м•ј.

A2. 8082 нҸ¬нҠём—җм„ң LISTENн•ҳкі  мһҲлҠ” н”„лЎңм„ёмҠӨк°Җ м—ҶлӢӨлҠ” лң»мқҙм•ј. мҰү, NexClass Spring Boot м• н”ҢлҰ¬мјҖмқҙм…ҳмқҙ мӢӨн–ү мӨ‘мқҙ м•„лӢҢ кұ°м§Җ. nohup java -jar NexClass.jar &лЎң мӢӨн–үн•ҙм•ј н•ҙ.

A3. м„ңлІ„ 1лҢҖм—җ н”„лЎңк·ёлһЁ 1к°ңл§Ң лҸҢлҰҙ мҲҳ мһҲм–ҙ. LMS, NexClass, HUBлҘј лҸҢлҰ¬л Өл©ҙ м„ңлІ„к°Җ 3лҢҖ н•„мҡ”н•ҳкІҢ лҸј. нҸ¬нҠёк°Җ мһҲмңјлӢҲк№Ң н•ҳлӮҳмқҳ IPм—җм„ң м—¬лҹ¬ н”„лЎңк·ёлһЁмқҙ к°Ғмһҗ лӢӨлҘё лІҲнҳёлЎң мҡ”мІӯмқ„ л°ӣмқ„ мҲҳ мһҲлҠ” кұ°м•ј. мһҗмӣҗ лӮӯл№„лҘј л§үм•„мЈјлҠ” н•өмӢ¬ к°ңл…җмқҙм§Җ.

A4. 443лІҲ нҸ¬нҠё. https://лҠ” HTTPS н”„лЎңнҶ мҪңмқҙкі , HTTPSмқҳ кё°ліё нҸ¬нҠёк°Җ 443мқҙм•ј. лёҢлқјмҡ°м Җк°Җ мһҗлҸҷмңјлЎң 443мқ„ л¶ҷм—¬м„ң м—°кІ°н•ҳкё° л•Ңл¬ём—җ URLм—җ м•Ҳ ліҙмқҙлҠ” кұ°м•ј. лӘ…мӢңм ҒмңјлЎң м“°л©ҙ https://nexclass.knu10.cc:443мқёлҚ°, кё°ліё нҸ¬нҠёлҠ” мғқлһөн•ҳлҠ” кІҢ кҙҖлЎҖм•ј.

A5. hostname -IлҠ” м„ңлІ„ мһҗкё° мһҗмӢ н•ңн…Ң "лӢҲ IP лӯҗм•ј?" л¬јм–ҙліҙлҠ” кұ°лқј лӮҙл¶Җ(мӮ¬м„Ө) IP(10.7.1.52)к°Җ лӮҳмҷҖ. curl ifconfig.meлҠ” мҷёл¶Җ м„ңл№„мҠӨн•ңн…Ң "лӮҙк°Җ м–ҙл””м„ң м ‘мҶҚн•ҳлҠ” кІғмІҳлҹј ліҙм—¬?" л¬јм–ҙліҙлҠ” кұ°лқј кіөмқё IP(49.247.45.181)к°Җ лӮҳмҷҖ. лӮҙл¶Җм—җм„ң ліҙлҠ” лӮҳ vs л°–м—җм„ң ліҙлҠ” лӮҳк°Җ лӢӨлҘё кұ°м•ј -- NAT(Network Address Translation) л•Ңл¬ёмқҙм•ј.
