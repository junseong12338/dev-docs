A1. Nginxмқҳ server_name(к°ҖмғҒ нҳёмҠӨнҢ…) л•Ңл¬ём—җ к°ҖлҠҘн•ҙ. DNSм—җм„ң 4к°ң лҸ„л©”мқё м „л¶Җ к°ҷмқҖ WEB м„ңлІ„ IPлЎң ліҙлӮҙлҸ„, HTTP мҡ”мІӯмқҳ Host н—ӨлҚ”м—җ лҸ„л©”мқё мқҙлҰ„мқҙ л“Өм–ҙмһҲм–ҙ. A2. мҡ°лҰ¬к°Җ мһҗмІҙ Sectigo SSL мқёмҰқм„ңлҘј Nginxм—җ м§Ғм ‘ м„Өм№ҳн•ҙм„ң м“°кі  мһҲкё° л•Ңл¬ёмқҙм•ј. Proxied лӘЁл“ңлЎң н•ҳл©ҙ Cloudflareк°Җ мӨ‘к°„м—җм„ң SSLмқ„ мІҳлҰ¬н•ҳл Ө н•ҙм„ң, мҡ°лҰ¬ мһҗмІҙ мқёмҰқм„ңмҷҖ м¶©лҸҢн•  мҲҳ мһҲм–ҙ. DNS OnlyлҠ” лӢЁмҲңнһҲ "мқҙ лҸ„л©”мқё = мқҙ IP" ліҖнҷҳл§Ң н•ҙмЈјкі , SSLмқҖ мҡ°лҰ¬ Nginxк°Җ м§Ғм ‘ мІҳлҰ¬н•ҙ.
