03. HTTPмҷҖ HTTPS - мҷң S н•ҳлӮҳк°Җ мӨ‘мҡ”н•ңк°Җ¶

лӮңмқҙлҸ„: Beta

01мһҘм—җм„ң IP/нҸ¬нҠё, 02мһҘм—җм„ң DNSлҘј л°°мӣ м–ҙ. мқҙм ң лҚ°мқҙн„°к°Җ м–ҙл–»кІҢ мҳӨк°ҖлҠ”м§Җ л°°мҡё м°ЁлЎҖм•ј. к·ёлҰ¬кі  мҷң Zoomмқҙ HTTPSл§Ң н—Ҳмҡ©н•ҳлҠ”м§Җ мқҙн•ҙн•  кұ°м•ј.

HTTPк°Җ лӯҗлғҗ¶

HTTPмқҳ ліём§Ҳ

HyperText Transfer Protocol -- лёҢлқјмҡ°м ҖмҷҖ м„ңлІ„к°Җ лҚ°мқҙн„°лҘј мЈјкі л°ӣлҠ” к·ңм№ҷ(н”„лЎңнҶ мҪң). "лӮҳ мқҙ нҺҳмқҙм§Җ мӨҳ" вҶ’ "м—¬кё° мһҲм–ҙ" мқҙ лҢҖнҷ”лҘј м–ҙл–Ө нҳ•мӢқмңјлЎң н• м§Җ м •н•ң кұ°м•ј.

HTTP мҡ”мІӯ-мқ‘лӢө лӘЁлҚё¶

sequenceDiagram
    participant Browser as лёҢлқјмҡ°м Җ
    participant Server as м„ңлІ„

    Browser->>Server: GET /health HTTP/1.1 (мҡ”мІӯ)
    Server-->>Browser: 200 OK {"status":"UP"} (мқ‘лӢө)
  • мҡ”мІӯ(Request): лёҢлқјмҡ°м Җк°Җ м„ңлІ„н•ңн…Ң "мқҙкұ° мӨҳ" лҳҗлҠ” "мқҙкұ° мІҳлҰ¬н•ҙ"
  • мқ‘лӢө(Response): м„ңлІ„к°Җ лёҢлқјмҡ°м Җн•ңн…Ң "м—¬кё° мһҲм–ҙ" лҳҗлҠ” "м—җлҹ¬м•ј"
  • нҸ¬нҠё: 80лІҲ (кё°ліё)

HTTPмқҳ л¬ём ң: нҸүл¬ё м „мҶЎ¶

HTTPмқҳ м№ҳлӘ…м Ғ м•Ҫм җ

HTTPлҠ” лҚ°мқҙн„°лҘј м•”нҳёнҷ” м—Ҷмқҙ к·ёлҢҖлЎң ліҙлӮҙ. мқҙкұё нҸүл¬ё(plaintext) м „мҶЎмқҙлқјкі  н•ҙ.

# HTTPлЎң ліҙлӮҙлҠ” лҚ°мқҙн„° (мӨ‘к°„м—җм„ң лҲ„к°Җ ліҙл©ҙ)
POST /webhook/zoom
Content-Type: application/json

{
  "event": "meeting.ended",
  "payload": {
    "meeting_id": 12345,
    "participants": ["к№ҖмІ мҲҳ", "мқҙмҳҒнқ¬", "л°•лҜјмҲҳ"]
  }
}
# вҶ‘ нҡҢмқҳ м°ёк°Җмһҗ м •ліҙк°Җ к·ёлҢҖлЎң л…ём¶ңлҗЁ!

мӨ‘к°„м—җ лҲ„к°Җ л„ӨнҠёмӣҢнҒ¬лҘј к°җмІӯн•ҳл©ҙ? лӢӨ ліҙм—¬. мқҙлҰ„, н•ҷлІҲ, нҡҢмқҳ м •ліҙ, м „л¶Җ лӢӨ.

HTTPSк°Җ лӯҗлғҗ¶

HTTPSмқҳ ліём§Ҳ

HTTP + SSL/TLS = HTTPS. HTTPм—җ м•”нҳёнҷ” кі„мёөмқ„ м¶”к°Җн•ң кұ°м•ј. лҚ°мқҙн„°к°Җ м•”нҳёнҷ”лҸјм„ң мӨ‘к°„м—җ лҲ„к°Җ лҙҗлҸ„ н•ҙлҸ… л¶Ҳк°ҖлҠҘ.

Zoomмқҙ мҷң HTTPSл§Ң н—Ҳмҡ©н•ҳлӮҳ¶

мқҙкІҢ мҳӨлҠҳ мҡ°лҰ¬к°Җ мӮҪм§Ҳн•ң к·јліё мӣҗмқёмқҙм•ј

Zoom Webhookмқҙ ліҙлӮҙлҠ” лҚ°мқҙн„°:

      "participants_count": 35

мқҙ лҚ°мқҙн„°м—җ нҡҢмқҳ м •ліҙ, м°ёк°Җмһҗ мҲҳ, к°•мқҳ мЈјм ңк°Җ л“Өм–ҙмһҲм–ҙ. HTTPлЎң ліҙлӮҙл©ҙ мӨ‘к°„м—җм„ң лӢӨ ліј мҲҳ мһҲм–ҙ.

к·ёлһҳм„ң ZoomмқҖ

  • Webhook URLм—җ HTTPSл§Ң н—Ҳмҡ©
  • HTTP URL л“ұлЎқн•ҳл©ҙ вҶ’ "URL validation failed" м—җлҹ¬
  • http://49.247.45.181:8082/webhook/zoom вҶ’ л“ұлЎқ м•Ҳ лҗЁ
  • https://nexclass.knu10.cc/webhook/zoom вҶ’ л“ұлЎқ лҗЁ

HTTP вҶ’ HTTPS лҰ¬лӢӨмқҙл үнҠё¶

мӢӨл¬ҙм—җм„ңлҠ” HTTP(80)лЎң л“Өм–ҙмҳӨл©ҙ мһҗлҸҷмңјлЎң HTTPS(443)лЎң ліҙлӮҙмӨҳ.

# HTTP(80)лЎң л“Өм–ҙмҳӨл©ҙ HTTPS(443)лЎң лҰ¬лӢӨмқҙл үнҠё
server {
    listen 80;                              # 80лІҲ нҸ¬нҠёлЎң л“Өм–ҙмҳӨлҠ” мҡ”мІӯ
    server_name nexclass.knu10.cc;          # nexclass лҸ„л©”мқёмқј л•Ң
    return 301 https://$host$request_uri;   # HTTPSлЎң мҳҒкө¬ лҰ¬лӢӨмқҙл үнҠё
}
# $host = nexclass.knu10.cc (мҡ”мІӯн•ң лҸ„л©”мқё)
# $request_uri = /webhook/zoom (мҡ”мІӯн•ң кІҪлЎң)
# 301 = мҳҒкө¬ мқҙлҸҷ (лёҢлқјмҡ°м Җк°Җ лӢӨмқҢл¶Җн„° л°”лЎң HTTPSлЎң к°җ)

301 vs 302

  • 301 (Moved Permanently): "мҳҒкө¬м ҒмңјлЎң мқҙмӮ¬н–Ҳм–ҙ. лӢӨмқҢл¶Җн„° HTTPSлЎң мҷҖ"
  • 302 (Found): "мһ„мӢңлЎң мқҙлҸҷмқҙм•ј. лӢӨмқҢм—җ лӢӨмӢң HTTPлЎң мҷҖлҸ„ лҸј"
  • HTTPвҶ’HTTPS лҰ¬лӢӨмқҙл үнҠёлҠ” 301 м“°лҠ” кІҢ л§һм•„.

мҷң лӮҙл¶ҖлҠ” HTTPк°Җ кҙңм°®лғҗ

лӮҙл¶Җл§қ(10.7.1.x)мқҖ н•ҷкөҗ л„ӨнҠёмӣҢнҒ¬ м•Ҳмқҙм•ј. мҷёл¶Җм—җм„ң м ‘к·ј л¶Ҳк°ҖлҠҘн•ҙ. м•”нҳёнҷ” мҳӨлІ„н—Өл“ң м—Ҷмқҙ л№ лҘҙкІҢ нҶөмӢ н•ҳлҠ” кІҢ лҚ” нҡЁмңЁм Ғмқҙм•ј. Nginxк°Җ HTTPS л°ӣм•„м„ң HTTPлЎң ліҖнҷҳн•ҙмЈјлҠ” м—ӯн• мқ„ н•ҳлҠ” кұ°м§Җ.

м •лҰ¬¶

к°ңл…җ н•ң мӨ„ м •лҰ¬
HTTP м•”нҳёнҷ” м—Ҷмқҙ лҚ°мқҙн„° мЈјкі л°ӣлҠ” н”„лЎңнҶ мҪң (нҸ¬нҠё 80)
HTTPS HTTP + SSL/TLS м•”нҳёнҷ” (нҸ¬нҠё 443)
нҸүл¬ё м „мҶЎ лҚ°мқҙн„°к°Җ м•”нҳёнҷ” м—Ҷмқҙ к·ёлҢҖлЎң м „мҶЎ (мӨ‘к°„мһҗ кіөкІ© м·Ём•Ҫ)
301 лҰ¬лӢӨмқҙл үнҠё HTTP вҶ’ HTTPS мҳҒкө¬ мқҙлҸҷ
лӮҙл¶Җ нҶөмӢ  к°ҷмқҖ л„ӨнҠёмӣҢнҒ¬л©ҙ HTTP OK (Nginx вҶ” NexClass)
мҷёл¶Җ нҶөмӢ  мқён„°л„· кІҪмң н•ҳл©ҙ HTTPS н•„мҲҳ (Zoom вҶ’ NexClass)

нҷ•мқё л¬ём ң¶

Q1. Zoom Marketplaceм—җм„ң webhook URLлЎң http://49.247.45.181:8082/webhook/zoomмқ„ л“ұлЎқн•ҳл Ө н–ҲлҚ”лӢҲ мӢӨнҢЁн–Ҳм–ҙ. мҷң?

Q2. мҡ°лҰ¬ Nginx м„Өм •м—җм„ң return 301 https://$host$request_uri;к°Җ н•ҳлҠ” мқјмқ„ м„ӨлӘ…н•ҙлҙҗ.

Q3. Nginx вҶ’ NexClass JAR кө¬к°„мқҖ HTTP(proxy_pass http://10.7.1.52:8082)мқёлҚ°, мқҙкІҢ ліҙм•ҲмғҒ л¬ём ң м—ҶлҠ” мқҙмң лҠ”?

Q4. HTTPSмқҳ кё°ліё нҸ¬нҠёк°Җ 443мқёлҚ°, мҡ°лҰ¬ NexClassлҠ” 8082м—җм„ң лҸҢм•„к°Җкі  мһҲм–ҙ. мқҙ м°ЁмқҙлҘј лҲ„к°Җ н•ҙкІ°н•ҙмЈјкі  мһҲм–ҙ?

м •лӢө ліҙкё°

A1. л‘җ к°Җм§Җ мқҙмң м•ј. (1) ZoomмқҖ HTTPSл§Ң н—Ҳмҡ©н•ҙ. HTTP URLмқҖ л“ұлЎқ мһҗмІҙк°Җ м•Ҳ лҸј. webhook лҚ°мқҙн„°м—җ нҡҢмқҳ м •ліҙ, м°ёк°Җмһҗ м •ліҙк°Җ нҸ¬н•ЁлҗҳлӢҲк№Ң м•”нҳёнҷ” н•„мҲҳ. (2) IP мЈјмҶҢ м§Ғм ‘ мӮ¬мҡ©мқҖ SSL мқёмҰқм„ң м Ғмҡ©мқҙ м–ҙл ӨмӣҢ. лҸ„л©”мқё кё°л°ҳмңјлЎң л“ұлЎқн•ҙм•ј н•ҙ.

A2. HTTP(80лІҲ нҸ¬нҠё)лЎң л“Өм–ҙмҳЁ мҡ”мІӯмқ„ HTTPS(443)лЎң мҳҒкө¬ лҰ¬лӢӨмқҙл үнҠён•ҳлҠ” кұ°м•ј. $hostлҠ” мҡ”мІӯн•ң лҸ„л©”мқё(nexclass.knu10.cc), $request_uriлҠ” кІҪлЎң(/webhook/zoom). мҰү http://nexclass.knu10.cc/webhook/zoom вҶ’ https://nexclass.knu10.cc/webhook/zoomмңјлЎң ліҙлӮҙмӨҳ. 301мқҙлӢҲк№Ң лёҢлқјмҡ°м Җк°Җ лӢӨмқҢл¶Җн„°лҠ” л°”лЎң HTTPSлЎң м ‘мҶҚн•ҙ.

A3. NginxмҷҖ NexClass JARмқҖ к°ҷмқҖ лӮҙл¶Җл§қ(10.7.1.x)м—җ мһҲм–ҙ. мҷёл¶Җм—җм„ң мқҙ кө¬к°„м—җ м ‘к·јн•  мҲҳ м—ҶмңјлӢҲк№Ң мӨ‘к°„мһҗ кіөкІ©мқҙ л¶Ҳк°ҖлҠҘн•ҙ. HTTPSмқҳ м•”нҳёнҷ” мҳӨлІ„н—Өл“ң м—Ҷмқҙ л№ лҘҙкІҢ нҶөмӢ н•ҳлҠ” кІҢ лҚ” нҡЁмңЁм Ғмқҙм•ј. SSLмқҖ Nginxк°Җ м•һлӢЁм—җм„ң мІҳлҰ¬н•ҳкі , лӮҙл¶ҖлЎңлҠ” HTTPлЎң м „лӢ¬н•ҳлҠ” нҢЁн„ҙмқ„ SSL мў…лЈҢ(SSL Termination)лқјкі  н•ҙ.

A4. Nginx(лҰ¬лІ„мҠӨ н”„лЎқмӢң)к°Җ н•ҙкІ°н•ҙмӨҳ. мҷёл¶Җм—җм„ң 443мңјлЎң л“Өм–ҙмҳЁ HTTPS мҡ”мІӯмқ„ Nginxк°Җ л°ӣм•„м„ң, SSL мІҳлҰ¬ нӣ„ лӮҙл¶Җ 8082 нҸ¬нҠёмқҳ NexClass JARлЎң HTTPлЎң м „лӢ¬н•ҙмӨҳ. мӮ¬мҡ©мһҗлҠ” 443л§Ң м•Ңл©ҙ лҗҳкі , мӢӨм ң м„ңл№„мҠӨ нҸ¬нҠё 8082лҠ” мҷёл¶Җм—җ л…ём¶ңлҗҳм§Җ м•Ҡм•„.
