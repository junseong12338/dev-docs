1. н”„лЎңм„ёмҠӨ л©”лӘЁлҰ¬ л Ҳмқҙм•„мӣғ - "мқҙкІҢ лӯҗм•ј?"В¶

нҡҢмӮ¬ мӮ¬л¬ҙмӢӨмқ„ мғқк°Ғн•ҙлҙҗ.

мӮ¬л¬ҙмӢӨ м•Ҳм—җ кө¬м—ӯмқҙ лӮҳлүҳм–ҙ мһҲмһ–м•„. нҡҢмқҳмӢӨ, к°ңмқё мұ…мғҒ, кіөмҡ© м°Ҫкі , м•ҲлӮҙ лҚ°мҠӨнҒ¬. к°Ғ кө¬м—ӯл§ҲлӢӨ мҡ©лҸ„к°Җ лӢӨлҘҙкі , м“°лҠ” л°©мӢқлҸ„ лӢӨлҘҙкі , лҲ„к°Җ кҙҖлҰ¬н•ҳлҠ”м§ҖлҸ„ лӢ¬лқј.

н”„лЎңм„ёмҠӨмқҳ л©”лӘЁлҰ¬лҸ„ кө¬м—ӯмқҙ лӮҳлүҳм–ҙ мһҲм–ҙ. к°Ғ кө¬м—ӯл§ҲлӢӨ м ҖмһҘн•ҳлҠ” лҚ°мқҙн„°к°Җ лӢӨлҘҙкі , кҙҖлҰ¬ л°©мӢқмқҙ лӢӨлҘҙкі , нҒ¬кё°к°Җ ліҖн•ҳлҠ” л°©мӢқлҸ„ лӢ¬лқј.

3. Stack мғҒм„ё - "н•өмӢ¬ мӨ‘мқҳ н•өмӢ¬"В¶

StackмқҙлһҖ?В¶

Stack = н•ЁмҲҳ нҳём¶ңкіј кҙҖл Ёлҗң лҚ°мқҙн„°лҘј м ҖмһҘн•ҳлҠ” LIFO(Last In, First Out) кө¬мЎ°мқҳ л©”лӘЁлҰ¬ мҳҒм—ӯ

LIFOк°Җ лӯҗлғҗ? м ‘мӢң мҢ“кё°м•ј. л§Ҳм§Җл§үм—җ мҳ¬лҰ° м ‘мӢңлҘј лЁјм Җ кәјлӮҙмһ–м•„. н•ЁмҲҳлҸ„ л§Ҳм°¬к°Җм§Җм•ј. л§Ҳм§Җл§үм—җ нҳём¶ңлҗң н•ЁмҲҳк°Җ лЁјм Җ лҒқлӮҳ.

н•ЁмҲҳк°Җ нҳём¶ңлҗ  л•Ңл§ҲлӢӨ мҠӨнғқ н”„л Ҳмһ„(Stack Frame)мқҙ н•ҳлӮҳм”© мҢ“м—¬. н•ЁмҲҳк°Җ лҒқлӮҳл©ҙ(return) к·ё н”„л Ҳмһ„мқҙ мӮ¬лқјм ё.

  1. мһҗлҸҷ кҙҖлҰ¬: н•ЁмҲҳ нҳём¶ң вҶ’ н”„л Ҳмһ„ мһҗлҸҷ мғқм„ұ / н•ЁмҲҳ return вҶ’ н”„л Ҳмһ„ мһҗлҸҷ м ңкұ°. к°ңл°ңмһҗк°Җ н• лӢ№/н•ҙм ң мӢ кІҪ м“ё н•„мҡ” м—ҶмқҢ
  2. л№ лҰ„: Stack Pointerл§Ң мқҙлҸҷн•ҳл©ҙ лҒқ (O(1)). нһҷмІҳлҹј л№Ҳ кіөк°„ м°ҫмқ„ н•„мҡ” м—ҶмқҢ
  3. нҒ¬кё° м ңн•ң: ліҙнҶө 1~8MB (OS/м–ём–ҙл§ҲлӢӨ лӢӨлҰ„). л¬ҙн•ңнһҲ мҢ“мңјл©ҙ вҶ’ Stack Overflow
  4. LIFO: л§Ҳм§Җл§үм—җ нҳём¶ңлҗң н•ЁмҲҳк°Җ лЁјм Җ мў…лЈҢ. нҳём¶ң мҲңм„ңмқҳ м—ӯмҲңмңјлЎң м •лҰ¬
  5. мҠӨл Ҳл“ңлі„: к°Ғ мҠӨл Ҳл“ңк°Җ мһҗкё°л§Ңмқҳ Stackмқ„ к°Җм§җ. лӢӨлҘё мҠӨл Ҳл“ңмқҳ Stackм—җ м ‘к·ј л¶Ҳк°Җ

4. Heap мғҒм„ё - "лҸҷм Ғ н• лӢ№мқҳ м„ёкі„"В¶

HeapмқҙлһҖ?В¶

Heap = н”„лЎңк·ёлһЁ мӢӨн–ү мӨ‘м—җ лҸҷм ҒмңјлЎң н• лӢ№/н•ҙм ңн•ҳлҠ” л©”лӘЁлҰ¬ мҳҒм—ӯ

StackмқҖ н•ЁмҲҳ нҳём¶ң мӢң мһҗлҸҷмңјлЎң кҙҖлҰ¬лҗҳм§Җл§Ң, HeapмқҖ к°ңл°ңмһҗк°Җ лӘ…мӢңм ҒмңјлЎң мҡ”мІӯн•ҙм•ј н•ҙ.

C/C++:   malloc() / free()          вҶҗ мҲҳлҸҷ кҙҖлҰ¬
Java:    new нӮӨмӣҢл“ң / GCк°Җ н•ҙм ң     вҶҗ GCк°Җ н•ҙм ң лӢҙлӢ№
Python:  к°қмІҙ мғқм„ұ / GCк°Җ н•ҙм ң      вҶҗ GCк°Җ н•ҙм ң лӢҙлӢ№

мҷң Heapмқҙ н•„мҡ”н•ҙ?В¶

Stackл§ҢмңјлЎң м¶©л¶„н•ҳм§Җ м•ҠмқҖ мғҒнҷ©мқҙ мһҲм–ҙ:

мҷң Heapмқҙ н•„мҡ”н•ҙ?

мғҒнҷ© Stack Heap
1. н•ЁмҲҳк°Җ лҒқлӮҳлҸ„ лҚ°мқҙн„°к°Җ мӮҙм•„мһҲм–ҙм•ј н•  л•Ң н•ЁмҲҳ returnн•ҳл©ҙ м§Җм—ӯліҖмҲҳ мҶҢл©ё н•ЁмҲҳ лҒқлӮҳлҸ„ к°қмІҙк°Җ мӮҙм•„мһҲмқҢ
2. нҒ¬кё°лҘј лҜёлҰ¬ лӘЁлҘј л•Ң м»ҙнҢҢмқј мӢң нҒ¬кё° кІ°м • лҹ°нғҖмһ„м—җ нҒ¬кё° кІ°м • к°ҖлҠҘ
3. нҒ° лҚ°мқҙн„° 1~8MB м ңн•ң GB лӢЁмң„ н• лӢ№ к°ҖлҠҘ
4. м—¬лҹ¬ мҠӨл Ҳл“ңк°Җ кіөмң н•ҙм•ј н•  л•Ң мҠӨл Ҳл“ңлі„ лҸ…лҰҪ лӘЁл“  мҠӨл Ҳл“ңк°Җ к°ҷмқҖ Heap кіөмң 

Heapмқҳ н•өмӢ¬ нҠ№м„ұВ¶

Heapмқҳ н•өмӢ¬ нҠ№м„ұ

  1. мҲҳлҸҷ кҙҖлҰ¬ (C/C++) лҳҗлҠ” GC кҙҖлҰ¬ (Java/Python): Cм—җм„ң free() м•Ҳ н•ҳл©ҙ л©”лӘЁлҰ¬ лҲ„мҲҳ. Javaм—җм„ң м°ёмЎ° м•Ҳ лҒҠмңјл©ҙ GCк°Җ н•ҙм ң лӘ» н•Ё
  2. лҠҗлҰј: л№Ҳ кіөк°„ м°ҫм•„м•ј н•Ё (л©”лӘЁлҰ¬ н• лӢ№ м•Ңкі лҰ¬мҰҳ). лӢЁнҺёнҷ” л°ңмғқ к°ҖлҠҘ
  3. нҒ¬кё° мң м—°: лҹ°нғҖмһ„м—җ нҒ¬кё° кІ°м •. н•„мҡ”н•ң л§ҢнҒј мҡ”мІӯ. GB лӢЁмң„лҸ„ к°ҖлҠҘ (RAM н—Ҳмҡ© лІ”мң„ лӮҙ)
  4. мҠӨл Ҳл“ң кіөмң : к°ҷмқҖ н”„лЎңм„ёмҠӨмқҳ лӘЁл“  мҠӨл Ҳл“ңк°Җ Heap кіөмң  вҶ’ лҸҷмӢң м ‘к·ј мӢң лҸҷкё°нҷ” н•„мҡ” (synchronized л“ұ)
  5. лӢЁнҺёнҷ” л¬ём ң: н• лӢ№/н•ҙм ң л°ҳліөн•ҳл©ҙ л№Ҳ кіөк°„мқҙ мЎ°к°ҒлӮЁ

5. Stack vs Heap л№„көҗ - "мқҙкұ° л©ҙм ‘м—җ лӮҳмҳЁлӢӨ"В¶

н•ӯлӘ© Stack Heap
м ҖмһҘ лҢҖмғҒ м§Җм—ӯліҖмҲҳ, л§Өк°ңліҖмҲҳ, л°ҳнҷҳ мЈјмҶҢ лҸҷм Ғ н• лӢ№ к°қмІҙ (new, malloc)
н• лӢ№ мҶҚлҸ„ л§Өмҡ° л№ лҰ„ (O(1)), SP мқҙлҸҷл§Ң н•ҳл©ҙ лҒқ лҠҗлҰј (л№Ҳ кіөк°„ нғҗмғү), н• лӢ№ м•Ңкі лҰ¬мҰҳ мӢӨн–ү
н•ҙм ң л°©мӢқ мһҗлҸҷ (н•ЁмҲҳ мў…лЈҢ) мҲҳлҸҷ (free) / GC
нҒ¬кё° мһ‘мқҢ (1~8MB) нҒј (мҲҳ GB к°ҖлҠҘ)
м„ұмһҘ л°©н–Ҙ лҶ’мқҖ мЈјмҶҢ вҶ’ лӮ®мқҖ мЈјмҶҢ (м•„лһҳлЎң) лӮ®мқҖ мЈјмҶҢ вҶ’ лҶ’мқҖ мЈјмҶҢ (мң„лЎң)
мҠӨл Ҳл“ң кіөмң  кіөмң  м•Ҳ н•Ё (мҠӨл Ҳл“ңлі„ лҸ…лҰҪ) кіөмң н•Ё (лҸҷкё°нҷ” н•„мҡ”)
мҲҳлӘ… н•ЁмҲҳ мҠӨмҪ”н”„ м°ёмЎ°к°Җ мһҲлҠ” лҸҷм•Ҳ мң м§Җ
лӢЁнҺёнҷ” м—ҶмқҢ л°ңмғқ к°ҖлҠҘ
мҳӨлІ„н”ҢлЎңмҡ° мӢң StackOverflowError OutOfMemoryError
кҙҖлҰ¬ мЈјмІҙ м»ҙнҢҢмқјлҹ¬/OS к°ңл°ңмһҗ/GC

6. мҷң Stackкіј Heapмқ„ л¶„лҰ¬н–ҲлҠ”к°Җ? - "5 Whys"В¶

мқҙкұҙ к·ёлғҘ л„ҳм–ҙк°Җл©ҙ м•Ҳ лҸј. к·јліёк№Ңм§Җ нҢҢм•ј н•ҙ.

Q1: "мҷң Stackкіј Heapмқ„ л¶„лҰ¬н–Ҳм–ҙ?"
A1: "мҡ©лҸ„к°Җ лӢӨлҘҙлӢҲк№Ңмҡ”."

Q2: "мҡ©лҸ„к°Җ лӢӨлҘҙлӢӨлҠ” кІҢ кө¬мІҙм ҒмңјлЎң лӯҗм•ј?"
A2: "StackмқҖ н•ЁмҲҳ нҳём¶ң кҙҖлҰ¬, HeapмқҖ лҸҷм Ғ лҚ°мқҙн„° кҙҖлҰ¬мҡ”."

Q3: "мҷң к°ҷмқҖ мҳҒм—ӯм—җм„ң л‘ҳ лӢӨ лӘ» н•ҙ?"
A3: "StackмқҖ LIFO мҲңм„ңлЎң л°ҳл“ңмӢң м •лҰ¬лҗҳм–ҙм•ј н•ҳлҠ”лҚ°,
     Heapмқҳ лҚ°мқҙн„°лҠ” мҲңм„ң м—Ҷмқҙ мғқм„ұ/мҶҢл©ёлҸјмҡ”.
     л‘ҳмқҳ мғқлӘ…мЈјкё° кҙҖлҰ¬ л°©мӢқмқҙ к·јліём ҒмңјлЎң лӢӨлҰ…лӢҲлӢӨ."

Q4: "мғқлӘ…мЈјкё° кҙҖлҰ¬ л°©мӢқмқҙ лӢӨлҘҙл©ҙ мҷң к°ҷмқҙ лӘ» н•ҙ?"
A4: "StackмқҖ SP(Stack Pointer) н•ҳлӮҳл§Ң мӣҖм§Ғмқҙл©ҙ O(1)лЎң кҙҖлҰ¬ к°ҖлҠҘн•ңлҚ°,
     HeapмІҳлҹј м•„л¬ҙ мҲңм„ңлЎң н• лӢ№/н•ҙм ңн•ҳл©ҙ лӢЁнҺёнҷ”к°Җ мғқкё°кі 
     л№Ҳ кіөк°„ нғҗмғүм—җ мҳӨлІ„н—Өл“ңк°Җ мғқкІЁмҡ”.
     Stackмқҳ O(1) м„ұлҠҘмқ„ мң м§Җн•ҳл Өл©ҙ л¶„лҰ¬н•ҙм•ј н•©лӢҲлӢӨ."

Q5: "к·ёлһҳм„ң к·јліё мӣҗм№ҷмқҙ лӯҗм•ј?"
A5: "мғқлӘ…мЈјкё°(lifetime)мҷҖ н• лӢ№ нҢЁн„ҙ(allocation pattern)мқҙ
     лӢӨлҘё лҚ°мқҙн„°лҠ” лі„лҸ„ мҳҒм—ӯм—җм„ң кҙҖлҰ¬н•ҙм•ј
     к°Ғ мҳҒм—ӯмқҳ мөңм Ғ кҙҖлҰ¬ м „лһөмқ„ м Ғмҡ©н•  мҲҳ мһҲлӢӨ.

     Stack: мҳҲмёЎ к°ҖлҠҘн•ң LIFO вҶ’ нҸ¬мқён„° мқҙлҸҷл§ҢмңјлЎң мҙҲкі мҶҚ кҙҖлҰ¬
     Heap: мҳҲмёЎ л¶Ҳк°ҖлҠҘн•ң лһңлҚӨ нҢЁн„ҙ вҶ’ ліөмһЎн•ң н• лӢ№мһҗ н•„мҡ”

     мқҙкұё м„һмңјл©ҙ л‘ҳ лӢӨ лҠҗл Өм§„лӢӨ. л¶„лҰ¬н•ҙм•ј к°Ғмһҗ мөңм Ғнҷ” к°ҖлҠҘ."

7. Stack Overflow vs OutOfMemoryError - "мҪ”л“ңлЎң ліҙмһҗ"В¶

Stack OverflowВ¶

// вқҢ мқҙлҹ¬л©ҙ н„°м§„лӢӨ
public class StackOverflowDemo {

    public static void infinite() {
        infinite();  // мһҗкё° мһҗмӢ мқ„ кі„мҶҚ нҳём¶ң (л¬ҙн•ң мһ¬к·Җ)
    }
    // нҳём¶ңн•  л•Ңл§ҲлӢӨ Stackм—җ н”„л Ҳмһ„мқҙ мҢ“мһ„

    }

OutOfMemoryErrorВ¶

// вқҢ мқҙлҹ¬л©ҙ н„°м§„лӢӨ
import java.util.ArrayList;
import java.util.List;

public class OOMDemo {

    public static void main(String[] args) {
        List<byte[]> list = new ArrayList<>();

        while (true) {
            list.add(new byte[1024 * 1024]);  // 1MB л°°м—ҙмқ„ Heapм—җ кі„мҶҚ н• лӢ№
            // listк°Җ м°ёмЎ°лҘј мң м§Җн•ҳкі  мһҲмңјлҜҖлЎң GCк°Җ н•ҙм ң лӘ» н•Ё
            // Heap нҒ¬кё° н•ңкі„ лҸ„лӢ¬
            // вҶ’ java.lang.OutOfMemoryError: Java heap space
        }
    }
}

8. Javaм—җм„ң Stackкіј Heap кө¬л¶„ - "мҪ”л“ңлЎң ліҙмһҗ"В¶

мқҙкІҢ к°ҖмһҘ мӨ‘мҡ”н•ҙ. лӯҗк°Җ Stackм—җ к°Җкі  лӯҗк°Җ Heapм—җ к°ҖлҠ”м§Җ нҷ•мӢӨнһҲ кө¬л¶„н•ҙм•ј н•ҙ.

public class MemoryDemo {

    // static ліҖмҲҳ вҶ’ Method Area (Data мҳҒм—ӯмқҳ мқјмў…)
    static int classVar = 100;

    public static void main(String[] args) {    // args вҶ’ Stack (main н”„л Ҳмһ„)

        // кё°ліёнҳ•(Primitive) вҶ’ Stack
        int x = 10;                             // Stack: x = 10
        double d = 3.14;                        // Stack: d = 3.14
        boolean flag = true;                    // Stack: flag = true

        // м°ёмЎ°нҳ•(Reference) вҶ’ м°ёмЎ°лҠ” Stack, к°қмІҙлҠ” Heap
        String s = new String("hello");         // Stack: s = 0xABC (м°ёмЎ°/мЈјмҶҢ)
                                                // Heap:  0xABC вҶ’ "hello" к°қмІҙ

        int[] arr = new int[5];                 // Stack: arr = 0xDEF (м°ёмЎ°)
                                                // Heap:  0xDEF вҶ’ [0,0,0,0,0]

        Person p = new Person("Kim", 25);       // Stack: p = 0x123 (м°ёмЎ°)
                                                // Heap:  0x123 вҶ’ Person к°қмІҙ
    }
}
