GCлҘј лӘЁлҘҙл©ҙ? - "мҷң к°‘мһҗкё° мқ‘лӢөмқҙ лҠҗл ӨмЎҢм–ҙмҡ”?" вҶ’ лӘЁлҰ„ - "Full GCк°Җ лңЁлҠ”лҚ° лӯ”к°Җмҡ”?" вҶ’ лӘЁлҰ„ - "GC лЎңк·ё н•ҙм„қн•ҙлҙҗ" вҶ’ лӘ» н•Ё

к·ёкұҙ мҡҙм „н•ҳл©ҙм„ң м—”м§„мқҙ лӯ”м§Җ лӘЁлҘҙлҠ” кұ°м•ј. нҸүмҶҢм—” кҙңм°®м•„. м—”м§„ н„°м§Ҳ л•Ңк№Ңм§Җ.

1. GCк°Җ лӯ”м§Җ - "м“°л Ҳкё° мҲҳкұ°"¶

л№„мң : м•„нҢҢнҠё м“°л Ҳкё° мҲҳкұ°¶

л№„мң : м•„нҢҢнҠё = JVM н”„лЎңм„ёмҠӨ

  • мЈјлҜј(м•ұ мҠӨл Ҳл“ң)мқҙ мқјмғҒмғқнҷңн•ҳл©ҙм„ң м“°л Ҳкё°(к°қмІҙ)лҘј л§Ңл“ лӢӨ.
  • м“°л Ҳкё°к°Җ мҢ“мқҙл©ҙ мҲҳкұ°мӣҗ(GC)мқҙ мҷҖм„ң м№ҳмҡҙлӢӨ.
  • к·јлҚ° мҲҳкұ°н•ҳл Өл©ҙ мЈјлҜјмқҙ мһ к№җ л°–м—җ лӮҳк°Җм•ј н•ҙ. вҶ’ мқҙкІҢ Stop-The-World (STW)
  • мқҙмӮҝм§җ(м°ёмЎ° мһҲлҠ” к°қмІҙ)мқҖ к°Җм ёк°Җл©ҙ м•Ҳ лҸј. вҶ’ мқҙкІҢ Reachability (лҸ„лӢ¬ к°ҖлҠҘм„ұ)
  • мҲҳкұ°мӣҗмқҙ мҳӨлһҳ мқјн•ҳл©ҙ? мЈјлҜјмқҙ мҳӨлһҳ лӘ» л“Өм–ҙк°Җм„ң мғқнҷң(м„ңл№„мҠӨ) л§Ҳл№„. вҶ’ мқҙкІҢ GC Pause (STW мӢңк°„)

л№„мң лҠ” мһ…кө¬кі , м§„м§ң м •мқҳ:

GC = лҚ” мқҙмғҒ мӮ¬мҡ©лҗҳм§Җ м•ҠлҠ”(лҸ„лӢ¬ л¶Ҳк°ҖлҠҘн•ң) к°қмІҙлҘј мһҗлҸҷмңјлЎң нғҗм§Җн•ҳкі  л©”лӘЁлҰ¬лҘј нҡҢмҲҳн•ҳлҠ” JVMмқҳ л©”м»ӨлӢҲмҰҳ

GCк°Җ м—Ҷмңјл©ҙ м–ҙл–»кІҢ лҗҳлғҗ?¶

C/C++ мІҳлҹј к°ңл°ңмһҗк°Җ м§Ғм ‘ malloc/free, new/delete н•ҙм•ј н•ңлӢӨ.

GC м—ҶлҠ” м„ёкі„мқҳ л¬ём ң:
1. free м•Ҳ н•ҳл©ҙ вҶ’ л©”лӘЁлҰ¬ лҲ„мҲҳ (Memory Leak)
2. мқҙлҜё freeн•ң кұё лҳҗ м“°л©ҙ вҶ’ Dangling Pointer (н”„лЎңк·ёлһЁ нҸӯл°ң)
3. к°ҷмқҖ кұё л‘җ лІҲ freeн•ҳл©ҙ вҶ’ Double Free (н”„лЎңк·ёлһЁ нҸӯл°ң)
4. м–ём ң freeн• м§Җ нҢҗлӢЁн•ҳлҠ” кІҢ вҶ’ мЎҙлӮҳ м–ҙл ӨмӣҖ (л©ҖнӢ°мҠӨл Ҳл“ңл©ҙ лҚ”)

Javaк°Җ GCлҘј лҸ„мһ…н•ң мқҙмң : к°ңл°ңмһҗк°Җ л©”лӘЁлҰ¬ н•ҙм ң мӢӨмҲҳн•ҳлҠ” кұё л§үкё° мң„н•ҙ. лҢҖмӢ  к·ё л№„мҡ©мқ„ GCк°Җ STWлЎң к°Җм ёк°„лӢӨ. мқҙкІғлҸ„ нҠёл Ҳмқҙл“ңмҳӨн”„м•ј.

2. м“°л Ҳкё°лҘј м–ҙл–»кІҢ м°ҫлӮҳ? - "мЈҪмқҖ к°қмІҙ нҢҗлі„лІ•"¶

мӣҗлҰ¬: к°қмІҙл§ҲлӢӨ "лӮҳлҘј к°ҖлҰ¬нӮӨлҠ” м°ёмЎ° мҲҳ"лҘј м„ёлӢӨ.
      м°ёмЎ° мҲҳк°Җ 0мқҙ лҗҳл©ҙ вҶ’ м“°л Ҳкё°

м№ҳлӘ…м Ғ лӢЁм җ: мҲңнҷҳ м°ёмЎ°лҘј лӘ» мһЎлҠ”лӢӨ.

Pythonмқҙ Reference Counting м“°лҠ”лҚ°, мҲңнҷҳ м°ёмЎ° н•ҙкІ° мң„н•ҙ лі„лҸ„ cycle detectorлҘј лҸҢлҰ°лӢӨ. JVMмқҖ Reference Countingмқ„ м•Ҳ м“ҙлӢӨ. лҢҖмӢ  м•„лһҳ л°©лІ•мқ„ м“ҙлӢӨ.

GC Rootм—җм„ң мӢңмһ‘н•ҙм„ң м°ёмЎ° мІҙмқёмқ„ л”°лқјк°Җл©° лҸ„лӢ¬ к°ҖлҠҘн•ң к°қмІҙлҘј м°ҫлҠ”лӢӨ. лҸ„лӢ¬ л¶Ҳк°ҖлҠҘн•ң к°қмІҙ вҶ’ м“°л Ҳкё°. мҲңнҷҳ м°ёмЎ°м—¬лҸ„ GC Rootм—җм„ң м•Ҳ лӢҝмңјл©ҙ м“°л Ҳкё°.

н•өмӢ¬: GC Rootм—җм„ң м°ёмЎ° мІҙмқёмқ„ л”°лқјк°Ҳ мҲҳ мһҲмңјл©ҙ мӮҙлҰј, лӘ» л”°лқјк°Җл©ҙ мЈҪмһ„. мҲңнҷҳ м°ёмЎ°? мғҒкҙҖм—ҶлӢӨ. GC Rootм—җм„ң м•Ҳ лӢҝмңјл©ҙ л‘ҳ лӢӨ м“°л Ҳкё°.

2.3 GC Root 4к°Җм§Җ¶

"GC Rootк°Җ лӯ”лҚ°?" вҶ’ мӮҙм•„мһҲлӢӨкі  ліҙмһҘлҗҳлҠ” м°ёмЎ°мқҳ мӢңмһ‘м җ.

GC Root 4к°Җм§Җ

  1. Stack м§Җм—ӯліҖмҲҳ (Local Variables): нҳ„мһ¬ мӢӨн–ү мӨ‘мқё л©”м„ңл“ңмқҳ м§Җм—ӯліҖмҲҳ, л§Өк°ңліҖмҲҳ. л©”м„ңл“ң лҒқлӮҳл©ҙ мҠӨнғқ н”„л Ҳмһ„ мӮ¬лқјм§Җл©ҙм„ң м°ёмЎ°лҸ„ мӮ¬лқјм§җ
  2. Static ліҖмҲҳ (Class Variables): нҒҙлһҳмҠӨм—җ мҶҚн•ҳлҠ” static н•„л“ң. нҒҙлһҳмҠӨк°Җ м–ёлЎңл”©лҗҳм§Җ м•ҠлҠ” н•ң кі„мҶҚ мӮҙм•„мһҲмқҢ. static м»¬л үм…ҳм—җ кі„мҶҚ л„Јмңјл©ҙ = лҲ„мҲҳ
  3. JNI м°ёмЎ° (Native Method References): JNIлЎң л„ӨмқҙнӢ°лёҢ мҪ”л“ңм—җм„ң м°ёмЎ°н•ҳлҠ” Java к°қмІҙ. л„ӨмқҙнӢ°лёҢ мҪ”л“ңк°Җ лӘ…мӢңм ҒмңјлЎң н•ҙм ңн•ҙм•ј н•Ё
  4. мӢӨн–ү мӨ‘мқё мҠӨл Ҳл“ң (Active Threads): Thread к°қмІҙ мһҗмІҙ + к·ё мҠӨл Ҳл“ңмқҳ мҠӨнғқм—җ мһҲлҠ” лӘЁл“  м°ёмЎ°. мҠӨл Ҳл“ңк°Җ мӮҙм•„мһҲмңјл©ҙ к·ё мҠӨл Ҳл“ңк°Җ м°ёмЎ°н•ҳлҠ” лӘЁл“  к°қмІҙлҸ„ мӮҙм•„мһҲмқҢ

Compact мӮҙм•„лӮЁмқҖ к°қмІҙлҘј м•һмңјлЎң мқҙлҸҷ, м°ёмЎ° к°ұмӢ  мӮҙм•„мһҲлҠ” к°қмІҙ мҲҳм—җ л№„лЎҖ (л№„мҢҲ!)

CompactлҘј мҷң н•ҳлғҗ? - Sweepл§Ң н•ҳл©ҙ л©”лӘЁлҰ¬ лӢЁнҺёнҷ”(Fragmentation) л°ңмғқ - л№Ҳ кіөк°„мқҙ м—¬кё°м Җкё° нқ©м–ҙм ём„ң нҒ° к°қмІҙ н• лӢ№ л¶Ҳк°Җ - CompactлЎң л°Җм–ҙм•ј м—°мҶҚ кіөк°„ нҷ•ліҙ к°ҖлҠҘ - лҢҖмӢ  CompactлҠ” к°қмІҙ мқҙлҸҷ + м°ёмЎ° мЈјмҶҢ к°ұмӢ мқҙлқј л№„мҡ©мқҙ нҒ¬лӢӨ

10. м •лҰ¬¶

н•ң мӨ„ м •лҰ¬¶

GC = GC Rootм—җм„ң лҸ„лӢ¬ л¶Ҳк°ҖлҠҘн•ң к°қмІҙлҘј м°ҫм•„ л©”лӘЁлҰ¬лҘј нҡҢмҲҳн•ҳлҠ” мһҗлҸҷ л©”м»ӨлӢҲмҰҳ. н•өмӢ¬мқҖ STW мөңмҶҢнҷ”.

н•ӯлӘ© н•өмӢ¬

кё°ліё лҸҷмһ‘ Mark(н‘ңмӢң) вҶ’ Sweep(м ңкұ°) вҶ’ Compact(м••м¶•)
Minor GC Young мҳҒм—ӯ, л№ лҰ„, мһҗмЈј, м •мғҒ
Full GC м „мІҙ Heap, лҠҗлҰј, к°ҖлҒ”, мһҗмЈј лңЁл©ҙ мң„н—ҳ
STW GC лҸҷм•Ҳ м•ұ л©Ҳм¶Ө. нҳ„лҢҖ GCмқҳ лӘ©н‘ңлҠ” STW мөңмҶҢнҷ”
м•Ңкі лҰ¬мҰҳ Serial вҶ’ Parallel вҶ’ CMS вҶ’ G1 вҶ’ ZGC (м җм җ STW мӨ„мһ„)
