Q26. API мқёмҰқкіј JWT (м„ңмҲ нҳ•, 4м җ)В¶

м •лӢө:

API мқёмҰқмқҖ APIлҘј нҳём¶ңн•ҳлҠ” мӘҪмқҙ "лӮҳ лҲ„кө¬м•ј"лҘј мҰқлӘ…н•ҳлҠ” кіјм •мқҙлӢӨ. л№„кіөк°ң APIлҠ” мқёк°Җлҗң мӢңмҠӨн…ңл§Ң нҳём¶ңн•  мҲҳ мһҲм–ҙм•ј н•ҳкё° л•Ңл¬ём—җ мқёмҰқмқҙ н•„мҲҳлӢӨ. JWTлҠ” м„ңлӘ…лҗң нҶ нҒ°мңјлЎң мӢ мӣҗмқ„ мҰқлӘ…н•ҳл©°, м„ңлІ„к°Җ м•„л¬ҙкІғлҸ„ м ҖмһҘн•ҳм§Җ м•Ҡкі лҸ„ кІҖмҰқн•  мҲҳ мһҲлӢӨ. нҠ№нһҲ м„ңлІ„вҶ”м„ңлІ„ нҶөмӢ м—җм„ңлҠ” лёҢлқјмҡ°м Җ/мҝ нӮӨк°Җ м—Ҷм–ҙм„ң м„ём…ҳмқ„ м“ё мҲҳ м—Ҷкё° л•Ңл¬ём—җ JWTк°Җ м Ғн•©н•ҳлӢӨ. NexClassвҶ’LMS нҳём¶ң мӢң JWTлҘј Authorization н—ӨлҚ”м—җ мІЁл¶Җн•ҳл©ҙ, LMSлҠ” м„ңлӘ…мқ„ кІҖмҰқн•ҙм„ң мқёк°Җлҗң мӢңмҠӨн…ңмқём§Җ нҷ•мқён•ңлӢӨ.

мұ„м җ кё°мӨҖ: API мқёмҰқ м •мқҳ(1м җ) + мҷң н•„мҡ”н•ңм§Җ(1м җ) + JWT мһ‘лҸҷ л°©мӢқ(1м җ) + м„ңлІ„вҶ”м„ңлІ„ нҠ№мҲҳм„ұ(1м җ). "мқҙкұ° л¶Ҳн•©кІ©" кё°мӨҖ: "JWTлҠ” ліҙм•Ҳмқҙ мўӢм•„м„ң м“ҙлӢӨ" к°ҷмқҖ м¶”мғҒм Ғ лӢөліҖ.

Q27. SECRET_KEY л…ём¶ң мң„н—ҳкіј кҙҖлҰ¬ (м„ңмҲ нҳ•, 4м җ)В¶

м •лӢө:

SECRET_KEYк°Җ л…ём¶ңлҗҳл©ҙ кіөкІ©мһҗк°Җ мң нҡЁн•ң JWTлҘј м§Ғм ‘ л§Ңл“Ө мҲҳ мһҲлӢӨ. л§ҢлЈҢ мӢңк°„лҸ„ л§ҲмқҢлҢҖлЎң м„Өм • к°ҖлҠҘн•ҳкі , мқёмҰқ мӢңмҠӨн…ңмқҙ мҷ„м „нһҲ лҡ«л Өм„ң 5л§Ң лӘ… к°ңмқём •ліҙм—җ м ‘к·јн•  мҲҳ мһҲлӢӨ. л”°лқјм„ң мҪ”л“ңм—җ н•ҳл“ңмҪ”л”©н•ҳл©ҙ м•Ҳ лҗҳкі , м„Өм • нҢҢмқј(application.properties)лЎң л¶„лҰ¬н•ҙм•ј н•ңлӢӨ. лҚ” м•Ҳм „н•ҳкІҢлҠ” нҷҳкІҪліҖмҲҳлӮҳ л№„л°Җ кҙҖлҰ¬ м„ңл№„мҠӨ(AWS Secrets Manager л“ұ)лҘј мӮ¬мҡ©н•ңлӢӨ. мҡ°лҰ¬ н”„лЎңм қнҠёлҠ” м„Өм • нҢҢмқј л¶„лҰ¬ + @Value мЈјмһ… л°©мӢқмқ„ мӮ¬мҡ©н•ңлӢӨ.

мұ„м җ кё°мӨҖ: л…ём¶ң мӢң мң„н—ҳ(1.5м җ) + кҙҖлҰ¬ л°©лІ• 2к°Җм§Җ мқҙмғҒ(1.5м җ) + мҡ°лҰ¬ н”„лЎңм қнҠё л°©мӢқ(1м җ). "мқҙкұ° л¶Ҳн•©кІ©" кё°мӨҖ: "мҪ”л“ңм—җ л„Јмңјл©ҙ м•Ҳ лҗңлӢӨ"л§Ң м“°кі  мҷңмқём§Җ м„ӨлӘ… м—ҶлҠ” кІҪмҡ°.

Q28. л¬ҙкІ°м„ұ vs кё°л°Җм„ұ (м„ңмҲ нҳ•, 5м җ)В¶

м •лӢө:

л¬ҙкІ°м„ұ(Integrity)мқҖ "лҚ°мқҙн„°к°Җ ліҖмЎ°лҗҳм§Җ м•Ҡм•ҳмқҢ"мқ„ ліҙмһҘн•ҳлҠ” кІғмқҙкі , кё°л°Җм„ұ(Confidentiality)мқҖ "лҚ°мқҙн„°лҘј н—Ҳк°Җлҗҳм§Җ м•ҠмқҖ мһҗк°Җ мқҪмқ„ мҲҳ м—ҶмқҢ"мқ„ ліҙмһҘн•ҳлҠ” кІғмқҙлӢӨ.

JWTм—җм„ң: - л¬ҙкІ°м„ұ: Signature(HMAC-SHA256)к°Җ лӢҙлӢ№. Payloadк°Җ ліҖмЎ°лҗҳл©ҙ м„ңлӘ…мқҙ л¶Ҳмқјм№ҳн•ҙм„ң к°җм§Җ к°ҖлҠҘ. н•ҳм§Җл§Ң Payload мһҗмІҙлҠ” лҲ„кө¬лӮҳ мқҪмқ„ мҲҳ мһҲмңјлҜҖлЎң кё°л°Җм„ұмқҖ м—ҶлӢӨ. - кё°л°Җм„ұ: JWT мһҗмІҙм—җлҠ” м—ҶлӢӨ. PayloadлҠ” Base64 мқёмҪ”л”©мқј лҝҗ. LMS мқ‘лӢөмқҳ кё°л°Җм„ұмқҖ AES-256мқҙ лӢҙлӢ№н•ңлӢӨ.

м •лҰ¬н•ҳл©ҙ JWTмқҳ м„ңлӘ…мқҖ "лӘ» л°”кҫёкІҢ" н•ҳлҠ” кІғмқҙм§Җ "лӘ» мқҪкІҢ" н•ҳлҠ” кІҢ м•„лӢҲлӢӨ.

мұ„м җ кё°мӨҖ: л¬ҙкІ°м„ұ м •мқҳ(1м җ) + кё°л°Җм„ұ м •мқҳ(1м җ) + JWTм—җм„ң л¬ҙкІ°м„ұ лӢҙлӢ№(1м җ) + JWTм—җ кё°л°Җм„ұ м—ҶмқҢ(1м җ) + AESк°Җ кё°л°Җм„ұ лӢҙлӢ№(1м җ). "мқҙкұ° л¶Ҳн•©кІ©" кё°мӨҖ: "JWTлҠ” м•”нҳёнҷ”лҗҳм–ҙ мһҲм–ҙм„ң кё°л°Җм„ұмқҙ мһҲлӢӨ."

вҶ’ 02_Alpha_м•”нҳёнҷ”_мқёмҪ”л”©_н•ҙмӢұ.md, 06_Gamma_AES256_м•”нҳёнҷ”.md м°ёмЎ°

Q29. л§ҢлЈҢ мӢңк°„ нҠёл Ҳмқҙл“ңмҳӨн”„ (м„ңмҲ нҳ•, 4м җ)В¶

м •лӢө:

5л¶„: нғҲм·ЁлҸјлҸ„ 5л¶„л§Ң мң нҡЁн•ҙм„ң ліҙм•Ҳмқҙ к°•н•ҳм§Җл§Ң, мһҗмЈј мһ¬л°ңкёүн•ҙм•ј н•ҙм„ң нҡЁмңЁмқҙ л–Ём–ҙм§„лӢӨ. кёҙ мһ‘м—… мӨ‘м—җ л§ҢлЈҢлҗҳл©ҙ мһ¬л°ңкёү лЎңм§Ғмқҙ н•„мҲҳлӢӨ.

24мӢңк°„: н•ң лІҲ л°ңкёүмңјлЎң мҳӨлһҳ мӮ¬мҡ© к°ҖлҠҘн•ҙм„ң нҺён•ҳм§Җл§Ң, нғҲм·Ёлҗҳл©ҙ 24мӢңк°„ лҸҷм•Ҳ л¬ҙл°©л№„ мғҒнғңлӢӨ.

мҡ°лҰ¬ н”„лЎңм қнҠёлҠ” 30л¶„мңјлЎң м„Өм •н–ҲлӢӨ. мқҙмң : лҸҷкё°нҷ” мһ‘м—… 1нҡҢк°Җ ліҙнҶө 5~15л¶„мқҙлқј 30л¶„мқҙл©ҙ м¶©л¶„н•ҳкі , нғҲм·ЁлҸјлҸ„ 30л¶„ нӣ„ мһҗлҸҷ л¬ҙнҡЁнҷ”лҗҳм–ҙ м Ғм Ҳн•ң к· нҳ•м җмқҙлӢӨ. м„ңлІ„вҶ”м„ңлІ„ нҶөмӢ мқҙлқј нҶ нҒ° мһ¬мғқм„ұ л№„мҡ©мқҙ кұ°мқҳ м—Ҷм–ҙм„ң м§§кІҢ мң м§Җн•ҙлҸ„ л¶ҖлӢҙмқҙ м—ҶлӢӨ.

мұ„м җ кё°мӨҖ: 5л¶„ мһҘлӢЁм җ(1м җ) + 24мӢңк°„ мһҘлӢЁм җ(1м җ) + мҡ°лҰ¬ н”„лЎңм қнҠё м„Өм •кіј мқҙмң (2м җ). "мқҙкұ° л¶Ҳн•©кІ©" кё°мӨҖ: нҠёл Ҳмқҙл“ңмҳӨн”„ м—Ҷмқҙ "м§§мқ„мҲҳлЎқ мўӢлӢӨ"л§Ң м“ҙ кІҪмҡ°.

вҶ’ 07_Delta_JWT_ліҙм•Ҳ_м„Өкі„.md 2м Ҳ, 6м Ҳ м°ёмЎ°
